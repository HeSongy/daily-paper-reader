## 摘要
具身智能（Embodied AI）正从孤立的感知或动作模块，迈向能够理解、基于目标进行规划、通过机器人本体执行动作、监控进展并从经验中持续改进的物理智能体。现有系统仅部分覆盖这一闭环：端到端策略能够生成动作，但通常缺乏空间推理、规划与执行评估能力；而机器人智能体系统虽然能够编排工具或专家模块，却无法学习共享表示。这种割裂限制了通用物理智能体人工智能（Physical Agentic AI）的发展。我们提出 ACE-Brain-0.5，一个统一的具身基础模型，将机器人智能组织为五个相互耦合的功能：空间感知、决策制定、具身交互、自我监控与自我改进。ACE-Brain-0.5 构建于 ACE-Brain-0 之上，后者建立了跨机器人平台共享的空间智能框架，而 ACE-Brain-0.5 则将以理解为中心的模型扩展为闭环基础模型。单一的 8B 主干网络实现了前四项功能：对物体及其可供性进行定位，对三维与自我中心空间关系进行推理，将指令分解为子目标，生成导航与操作动作，并估计任务进展以进行验证与恢复。为在不同任务之间统一这些能力而避免交叉任务干扰，我们提出 SSR+，其在任务向量合并之后，为 Scaffold-Specialize-Reconcile 框架增加了一个 Reactivate 阶段。第五项功能——自我改进——通过一个配套框架实现，该框架能够基于 rollout 更新外部执行状态，包括任务模式、空间记忆以及失败恢复案例。在十五项基准测试中，ACE-Brain-0.5 在 18 个空间感知与定位基准中的 14 项上优于 ACE-Brain-0，在导航与操作任务中取得了具有竞争力的性能，并在 ID 与 OOD 场景下展现出强大的进展估计能力。这些结果共同标志着迈向通用物理智能体人工智能的重要早期一步。

## Abstract
Embodied AI is moving from isolated perception or action modules toward physical agents that understand, plan under goals, act through robot bodies, monitor progress, and improve from experience. Existing systems address this loop only in parts: end-to-end policies generate actions but often lack spatial reasoning, planning, and execution assessment, while robot-agent systems orchestrate tools or specialists but do not learn a shared representation. This fragmentation limits general Physical Agentic AI. We present ACE-Brain-0.5, a unified embodied foundation model that organizes robot intelligence into five coupled functions: spatial perception, decision making, embodied interaction, self-monitoring, and self-improvement. Built on ACE-Brain-0, which established spatial intelligence as a shared scaffold across robot platforms, ACE-Brain-0.5 extends an understanding-centric model into a closed-loop foundation model. A single 8B backbone instantiates the first four functions: grounding objects and affordances, reasoning over 3D and egocentric spatial relations, decomposing instructions into subgoals, generating navigation and manipulation actions, and estimating progress for verification and recovery. To unify these capabilities without cross-task interference, we introduce SSR+, which extends Scaffold-Specialize-Reconcile with a Reactivate stage after task-vector merging. The fifth function, self-improvement, is realized by a companion framework that updates external execution state, including task schemas, spatial memory, and failure-recovery cases, from rollouts. Across fifteen benchmarks, ACE-Brain-0.5 improves over ACE-Brain-0 on 14 of 18 spatial perception and grounding benchmarks, achieves competitive navigation and manipulation performance, and provides strong progress estimation in ID and OOD settings. Together, these results mark an early step toward general Physical Agentic AI.

模型直接输出：

- navigation action
- manipulation chunk

包含：

- 连续动作
- 离散动作

使用 flow-matching action expert：

- 从噪声初始化动作；
- 基于 decoder context 逐步生成动作块。

这类似 diffusion-style action policy。

---

### (4) 自我监控（重点）

这是论文一个重要创新。

模型能够预测：

- 当前任务完成度（progress score）

论文展示：

- 成功轨迹
- 失败轨迹
- 回退轨迹

progress curve 都较稳定。

其作用：

- verification
- recovery
- reward modeling

即：

模型不仅会做，还会判断“做得怎么样”。

---

### (5) 自我改进

并未完全内化到模型参数。

论文采用：

- 外部状态更新框架（external execution state）

维护：

- task schema
- spatial memory
- failure-recovery cases

通过 rollout 持续更新。

本质：

- 类似长期记忆系统；
- 形成闭环学习。

---

# 2.3 SSR+ 训练框架

这是论文方法层面的核心技术。

## 问题

多任务融合时：

- 空间推理
- 导航
- manipulation
- progress estimation

容易出现：

- task interference
- catastrophic forgetting

---

## 原始 SSR

已有 Scaffold-Specialize-Reconcile：

1. Scaffold
   - 建立共享基础能力

2. Specialize
   - 各任务单独训练

3. Reconcile
   - 合并 task vector

---

## SSR+ 新增 Reactivate

论文新增：

4. Reactivate

即：

在 task-vector merge 后，

重新激活：

- 原始 VLM 能力
- 多模态推理能力

避免：

- policy learning 破坏 foundation model 能力。

这是论文较重要的方法贡献。

---

## 2.4 Attention Mask 设计

模型使用混合 attention：

### VLM tokens

- causal attention

保持：

- 自回归 reasoning 能力

---

### Action tokens

- full attention

用于：

- 时空特征聚合
- 动作生成

这种 hybrid attention：

兼顾：

- 大模型 reasoning
- 高效控制策略学习

“空间理解”是具身智能统一基础。

# 8. 不足与局限

## 1. 自我改进仍较弱

“Self-improvement”：

更多依赖：

- 外部 memory
- rollout update

而不是：

- 真正 online parameter adaptation。

因此：

还不是真正 autonomous continual learner。

---

## 2. 真实机器人验证有限

缺少：

- 大规模真实机器人 deployment；
- 长周期稳定性；
- 长时间交互实验。

目前更多是 benchmark 驱动。

---

## 3. 统一模型可能存在性能折中

虽然统一性强：

但：

- 专用 manipulation policy
- 专用 navigation model

在单任务上可能仍更强。

论文未完全分析 trade-off。

---

## 4. 算力与数据透明度不足

缺少：

- 完整训练细节；
- 数据清洗策略；
- GPU 资源；
- 训练成本。

可复现性有限。

---

## 5. 闭环能力尚不完整

当前仍缺：

- 真正长期记忆学习；
- 世界模型推演；
- 强因果规划；
- 多轮交互学习；
- 主动探索。

因此：

更像“统一能力集合”，
而非完整 autonomous embodied AGI。

---

# 总结

ACE-Brain-0.5 的核心贡献在于：

- 首次较系统地提出“统一具身基础模型”框架；
- 用单一 8B backbone 融合：
  - 空间理解
  - 规划
  - 控制
  - 进度评估
- 并通过 SSR+ 缓解多任务冲突。

其真正重要的意义并不只是 benchmark 提升，而是：

它试图回答：

“机器人是否能够像 LLM 一样拥有统一大脑？”

论文给出的答案是：

至少在 early-stage closed-loop embodied intelligence 上，已经出现可行路径。
