## 摘要
具身智能（Embodied AI）正在从孤立的感知或动作模块，迈向能够理解、基于目标进行规划、通过机器人身体执行动作、监控进展并从经验中改进的物理智能体。现有系统仅部分覆盖这一闭环：端到端策略能够生成动作，但通常缺乏空间推理、规划与执行评估能力；而机器人智能体系统虽然能够编排工具或专用模块，却无法学习共享表征。这种碎片化限制了通用物理智能体人工智能（Physical Agentic AI）的发展。我们提出 ACE-Brain-0.5，一个统一的具身基础模型，将机器人智能组织为五个相互耦合的功能：空间感知、决策制定、具身交互、自我监控与自我改进。ACE-Brain-0.5 构建于 ACE-Brain-0 之上，后者将空间智能确立为跨机器人平台的共享支架，而 ACE-Brain-0.5 则进一步将以理解为中心的模型扩展为闭环基础模型。单一的 8B 主干网络实现了前四项功能：对物体及其可供性进行 grounding、基于三维与第一视角空间关系进行推理、将指令分解为子目标、生成导航与操作动作，并估计任务进展以支持验证与恢复。为了在统一这些能力的同时避免跨任务干扰，我们提出 SSR+，在任务向量合并后于 Scaffold-Specialize-Reconcile 框架中新增 Reactivate 阶段。第五项功能——自我改进——通过一个配套框架实现，该框架能够根据 rollout 更新外部执行状态，包括任务模式、空间记忆以及失败恢复案例。在十五项基准测试中，ACE-Brain-0.5 在 18 个空间感知与 grounding 基准中的 14 个上超越了 ACE-Brain-0，在导航与操作任务中取得了具有竞争力的性能，并在 ID 与 OOD 场景下展现出强大的进度估计能力。这些结果共同标志着迈向通用物理智能体人工智能的重要早期一步。

## Abstract
Embodied AI is moving from isolated perception or action modules toward physical agents that understand, plan under goals, act through robot bodies, monitor progress, and improve from experience. Existing systems address this loop only in parts: end-to-end policies generate actions but often lack spatial reasoning, planning, and execution assessment, while robot-agent systems orchestrate tools or specialists but do not learn a shared representation. This fragmentation limits general Physical Agentic AI. We present ACE-Brain-0.5, a unified embodied foundation model that organizes robot intelligence into five coupled functions: spatial perception, decision making, embodied interaction, self-monitoring, and self-improvement. Built on ACE-Brain-0, which established spatial intelligence as a shared scaffold across robot platforms, ACE-Brain-0.5 extends an understanding-centric model into a closed-loop foundation model. A single 8B backbone instantiates the first four functions: grounding objects and affordances, reasoning over 3D and egocentric spatial relations, decomposing instructions into subgoals, generating navigation and manipulation actions, and estimating progress for verification and recovery. To unify these capabilities without cross-task interference, we introduce SSR+, which extends Scaffold-Specialize-Reconcile with a Reactivate stage after task-vector merging. The fifth function, self-improvement, is realized by a companion framework that updates external execution state, including task schemas, spatial memory, and failure-recovery cases, from rollouts. Across fifteen benchmarks, ACE-Brain-0.5 improves over ACE-Brain-0 on 14 of 18 spatial perception and grounding benchmarks, achieves competitive navigation and manipulation performance, and provides strong progress estimation in ID and OOD settings. Together, these results mark an early step toward general Physical Agentic AI.

## 2.3 Fast Vision Encoder

为了解决机器人控制中的低延迟问题：

作者增加了：

- Fast Vision Encoder（DINOv3）

特点：

- 只处理最近 observation；
- 低延迟；
- 专门为动作控制提供实时特征。

这些特征直接输入：

- flow-matching Action Expert。

---

## 2.4 Action Expert

动作模块采用：

- Flow Matching 风格 action generation。

其功能：

- 从 noise initialized action 出发；
- 在高层语义条件下逐步积分；
- 输出：
  - 连续控制动作；
  - 离散导航动作。

支持：

- manipulation
- navigation
- hybrid action chunk

---

## 2.5 统一输出空间

模型统一生成四类输出：

### 1）空间输出

- bbox
- segmentation mask
- affordance grounding

### 2）规划输出

- 子目标（subgoals）
- chain-of-thought style decomposition

### 3）交互输出

- 导航动作
- manipulation trajectory

### 4）监控输出

- task progress estimation

形成：

Perception → Planning → Action → Evaluation 闭环。

---

## 2.6 SSR+ 训练机制

这是论文最核心的方法创新之一。

### 问题

多任务统一训练容易产生：

- task interference（任务干扰）
- catastrophic forgetting
- optimization conflict

---

### 原始 SSR

作者基于此前：

Scaffold-Specialize-Reconcile（SSR）

包括：

1. Scaffold
   - 学共享基础能力
2. Specialize
   - 针对任务独立适配
3. Reconcile
   - 合并任务向量

---

### SSR+ 的新增阶段

新增：

4. Reactivate

作用：

- 在任务向量 merge 后重新激活通用能力；
- 修复任务融合后的能力退化。

本质上：

SSR+ = 多任务 task-vector merge + 能力恢复机制。

---

## 2.7 Self-Improvement 机制

ACE-Brain-0.5 没有完全把“长期记忆”内化到参数中。

而是：

通过外部 memory framework 实现。

更新内容包括：

- task schema
- spatial memory
- failure-recovery cases

系统可基于 rollout：

- 更新执行经验；
- 增强恢复能力；
- 提升长期任务稳定性。

# 8. 不足与局限

## 8.1 仍未实现真正长期 agent

虽然提出：

self-improvement。

但：

- 主要依赖外部 memory；
- 不是 end-to-end continual learning。

长期自主学习能力仍有限。

---

## 8.2 缺少真实机器人大规模验证

实验大量基于：

- benchmark
- simulator

真实世界部署有限。

泛化能力仍待验证。

---

## 8.3 训练成本较高

统一模型需要：

- 多模态；
- 多任务；
- 长视频；
- action learning。

实际部署门槛高。

---

## 8.4 可解释性有限

虽然有 progress prediction，

但：

- 内部 planning mechanism
- latent world model

仍然较黑箱。

---

## 8.5 任务统一仍可能存在负迁移

SSR+ 虽然缓解冲突，

但：

- perception
- action
- language reasoning

之间的梯度竞争问题并未根本解决。

---

## 8.6 缺少 scaling law 分析

论文没有深入分析：

- 模型规模影响；
- 数据规模影响；
- 不同任务配比影响。

因此：

尚不清楚其未来扩展规律。

---

# 总结

ACE-Brain-0.5 的核心贡献在于：

- 将空间感知、规划、动作生成、执行监控、自我改进统一到单一 embodied foundation model 中；
- 通过 SSR+ 缓解多任务统一训练冲突；
- 强调“空间智能”作为通用机器人 cognition 的共享基础。

相比传统 VLA 或机器人 agent：

它更接近“统一机器人大脑”的概念。

虽然距离真正通用 Physical Agentic AI 仍有明显差距，但论文在：

- 闭环机器人认知、
- self-monitoring、
- unified embodied representation

方面提供了较有代表性的早期框架。
