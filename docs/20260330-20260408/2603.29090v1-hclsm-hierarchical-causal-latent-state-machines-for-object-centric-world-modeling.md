## Abstract
World models that predict future states from video remain limited by flat latent representations that entangle objects, ignore causal structure, and collapse temporal dynamics into a single scale. We present HCLSM, a world model architecture that operates on three interconnected principles: object-centric decomposition via slot attention with spatial broadcast decoding, hierarchical temporal dynamics through a three-level engine combining selective state space models for continuous physics, sparse transformers for discrete events, and compressed transformers for abstract goals, and causal structure learning through graph neural network interaction patterns. HCLSM introduces a two-stage training protocol where spatial reconstruction forces slot specialization before dynamics prediction begins. We train a 68M-parameter model on the PushT robotic manipulation benchmark from the Open X-Embodiment dataset, achieving 0.008 MSE next-state prediction loss with emerging spatial decomposition (SBD loss: 0.0075) and learned event boundaries. A custom Triton kernel for the SSM scan delivers 38x speedup over sequential PyTorch. The full system spans 8,478 lines of Python across 51 modules with 171 unit tests. ## 一、研究动机与背景

- **核心问题**：  
  现有视频预测与世界建模方法（如 V-JEPA、DreamerV3 等）通常依赖“平坦潜变量表示”（flat latent representation）。这种表示方式会将场景中所有对象、时间尺度与因果关系混合在一个潜空间内，导致：
  - 对象间语义无法独立建模；
  - 时序动态被压缩为单一时间尺度；
  - 模型难以回答诸如“如果某个因素变化会怎样”的因果性问题。
- **研究动机**：  
  作者认为真正的世界模型应当具备三种结构性能力：
  1. **对象中心性（object-centric）**：识别并跟踪独立对象；
  2. **分层时间结构（hierarchical temporal dynamics）**：适应不同时间尺度的物理过程与抽象目标；
  3. **因果推理（causal reasoning）**：显式或隐式地学习对象之间的因果关系。  
  因此，本文提出 **HCLSM（Hierarchical Causal Latent State Machines）**，在单一可微框架中整合这三种要素。

## 二、方法论概述

### 1. 核心思想

HCLSM旨在构建一个**对象级的分层世界模型**，利用视觉输入自动：
- 分解场景中的对象；
- 分别建模其连续与离散动态；
- 学习对象间的交互因果关系。

### 2. 模型总体结构（五层架构）

1. **感知层（Perception）**  
   使用 Vision Transformer (ViT) 提取视频片段的时空特征，并映射到统一维度 `d_world`。
2. **对象分解层（Object Decomposition）**  
   - 采用 Slot Attention 机制进行对象分组；  
   - 引入 **空间广播解码器（Spatial Broadcast Decoder, SBD）**，通过重建冻结的 ViT 特征来迫使每个 slot 专注于局部区域；
   - 设置“slot 诞生/死亡”机制，使模型可动态调整活跃对象数量；
   - 利用 **图神经网络（GNN）** 处理对象间关系，并输出交互权重作为因果信号。
3. **分层动态层（Hierarchical Dynamics）**  
   三个时序层共同作用：
   - **Level 0**：选择性状态空间模型（Selective SSM）模拟连续的物理动态；
   - **Level 1**：稀疏 Transformer 捕捉离散事件（如碰撞）；
   - **Level 2**：压缩 Transformer 聚合高层语义（如任务或目标）。
4. **因果结构层（Causal Structure）**  
   - 学习邻接矩阵 \(W \in \mathbb{R}^{N×N}\)；
   - 使用 Gumbel-softmax 进行可微化二值采样；
   - 应用 NOTEARS 约束确保图结构为有向无环图（DAG）。
5. **持续记忆层（Continual Memory）**  
   使用 Hopfield 网络 + EWC 约束实现长期记忆保持。

### 3. 两阶段训练流程

作者强调“**结构先于预测（structure before prediction）**”的原则：
1. **阶段一（Stage 1）**：仅优化空间重建损失 `L_SBD` 与多样性正则，用以逼迫 slots 实现对象分工；
2. **阶段二（Stage 2）**：在固定分解结构后，引入 JEPA 风格的动态预测损失 `L_JEPA` 与因果正则。

该过程解决了直接联合优化导致 slots 不专化的问题。

### 4. 算法实现亮点

- 提供 **Triton 自定义 SSM 内核**，使并行扫描比 PyTorch 顺序实现加速 38 倍；
- 在 bf16 精度下通过多项数值技巧（激活截断、改写梯度运算等）保持稳定；
- GPU 原生的 Slot 匹配（Sinkhorn-Knopp）替代 CPU 匈牙利算法。
