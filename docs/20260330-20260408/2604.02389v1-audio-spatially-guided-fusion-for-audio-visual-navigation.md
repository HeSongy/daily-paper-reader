## 摘要
音视频导航是指智能体在复杂的三维环境中利用视觉和听觉信息进行目标定位和路径规划，从而实现自主导航。该任务的核心挑战在于：如何让智能体摆脱对训练数据的依赖，在面对环境和声源变化时仍能实现具有良好泛化能力的自主导航。为解决这一问题，我们提出了一种用于音视频导航的音频空间引导融合方法。首先，我们设计了一个音频空间特征编码器，该编码器通过音频强度注意力机制自适应地提取与目标相关的空间状态信息；在此基础上，我们引入了音频空间状态引导融合（ASGF），实现了多模态特征的动态对齐与自适应融合，有效缓解了由感知不确定性带来的噪声干扰。在 Replica 和 Matterport3D 数据集上的实验结果表明，我们的方法在未见任务上表现尤为出色，在未知声源分布下展现出更强的泛化能力。

## Abstract
Audio-visual Navigation refers to an agent utilizing visual and auditory information in complex 3D environments to accomplish target localization and path planning, thereby achieving autonomous navigation. The core challenge of this task lies in the following: how the agent can break free from the dependence on training data and achieve autonomous navigation with good generalization performance when facing changes in environments and sound sources. To address this challenge, we propose an Audio Spatially-Guided Fusion for Audio-Visual Navigation method. First, we design an audio spatial feature encoder, which adaptively extracts target-related spatial state information through an audio intensity attention mechanism; based on this, we introduce an Audio Spatial State Guided Fusion (ASGF) to achieve dynamic alignment and adaptive fusion of multimodal features, effectively alleviating noise interference caused by perceptual uncertainty. Experimental results on the Replica and Matterport3D datasets indicate that our method is particularly effective on unheard tasks, demonstrating improved generalization under unknown sound source distributions.

## 二、方法论概述

### 1. 核心思想  
论文提出一种名为 **Audio Spatially-Guided Fusion for Audio-Visual Navigation (ASGF-Nav)** 的框架：  
通过**音频空间状态（Audio Spatial State）**引导视觉与音频特征融合，从而实现动态自适应的多模态决策。其两大核心模块为：

- **音频空间状态编码器（Audio Spatial State Encoder, ASE）**：  
  从**双耳频谱图（binaural spectrograms）**中提取**时空关联信息**，并通过音频强度注意力机制（Audio Intensity Attention）强化空间相关帧。
- **音频空间状态引导融合模块（Audio Spatial State-Guided Fusion, ASGF）**：  
  以音频空间状态为 Query，对视觉–听觉联合特征进行**跨模态注意力**与**自适应门控融合**。

### 2. 关键技术与流程

ASGF-Nav 的结构如下：

1. **输入**：RGB + 深度图（视觉），双耳音频信号（听觉）。
2. **特征提取**：
   - 视觉、听觉分别经 CNN 生成特征向量；
   - 听觉信号经 ASE 模块提取空间嵌入 `e_t`：
     - 音频帧强度 \( I_{t,i} = \mathrm{mean}_{f,c} X_t(f,i,c) \)
     - 归一化注意力权重 \( w_{t,i} = \mathrm{softmax}(I_{t,i}) \)
     - 加权聚合得到音频空间状态 \( e_t = \sum_i w_{t,i} \tilde{e}_{t,i} \)
3. **跨模态融合（ASGF 模块）**：
   - 构造跨模态记忆 \( X_t = [W_d d_t, W_a a_t] \)
   - 生成 Key/Value，并以音频空间状态 \( e_t \) 为 Query 进行注意力聚合：
     \( c_t = \mathrm{Attn}(W_q e_t, X_t W_K, X_t W_V) \)
   - 残差连接与门控机制联合输出：
     \( \hat{h}_t = c_t + e_t \)，\( h_t = \mathrm{LayerNorm}(\alpha_t \hat{h}_t + (1 - \alpha_t)e_t) \)
4. **策略学习**：  
   通过 GRU 融合时序历史状态，并输入 Actor–Critic 网络以输出动作策略（四类离散动作）。
