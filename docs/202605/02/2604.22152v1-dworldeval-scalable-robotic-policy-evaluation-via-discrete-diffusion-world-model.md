## 摘要
在数千种环境和数千项任务中评估机器人策略，用现有方法几乎不可行。这促使我们提出一种可扩展的机器人策略评估新方法。本文提出了 dWorldEval，它使用离散扩散世界模型作为可扩展的机器人策略评估代理。具体而言，dWorldEval 将所有模态——包括视觉、语言和机器人动作——映射到统一的 token 空间，并通过一个基于 Transformer 的去噪网络进行建模。基于这一架构，我们引入稀疏关键帧记忆以保持时空一致性。同时，我们还设计了一个表示任务完成度的进度 token。在推理阶段，模型联合预测未来观察值和进度 token，当进度达到 1 时可自动判定任务成功。大量实验表明，dWorldEval 在 LIBERO、RoboTwin 及多个真实机器人任务上显著优于以往方法，如 WorldEval、Ctrl-World 和 WorldGym。该研究为构建用于大规模机器人评估的世界模拟器开创了新的架构范式。

## Abstract
Evaluating robotics policies across thousands of environments and thousands of tasks is infeasible with existing approaches. This motivates the need for a new methodology for scalable robotics policy evaluation. In this paper, we propose dWorldEval, which uses a discrete diffusion world model as a scalable evaluation proxy for robotics policies. Specifically, dWorldEval maps all modalities - including vision, language, and robotic actions - into a unified token space, modeling them via a single transformer-based denoising network. In this paper, we propose dWorldEval, using a discrete diffusion world model as a scalable evaluation proxy for robotics policy. Specifically, it maps all modalities, including vision, language, and robotics action into a unified token space, then denoises them with a single transformer network. Building on this architecture, we employ a sparse keyframe memory to maintain spatiotemporal consistency. We also introduce a progress token that indicates the degree of task completion. At inference, the model jointly predicts future observations and progress token, allowing automatically determine success when the progress reaches 1. Extensive experiments demonstrate that dWorldEval significantly outperforms previous approaches, i.e., WorldEval, Ctrl-World, and WorldGym, on LIBERO, RoboTwin, and multiple real-robot tasks. It paves the way for a new architectural paradigm in building world simulators for robotics evaluation at scale.

- **研究目标**：构建一个统一、可控、可扩展的世界模型，使其不仅能生成视觉上连续的视频，还能**忠实地响应输入动作，并自动判断任务完成度**，从而高效且可靠地评估机器人策略。

## 二、方法论：dWorldEval 架构与关键机制

### 1. 核心思想

> 将视觉、语言、动作三种模态统一映射到同一个**离散 token 空间**，通过单一 Transformer 去噪网络建模未来状态的条件分布。

### 2. 模型核心组件

- **统一 token 序列**（Unified Token Sequence）
  - 视觉：使用 MAGVIT-v2 编码为离散图像 token  
  - 语言：使用 LLaDA 语言编码器  
  - 动作：用 FAST 编码连续动作向量  
  - 三者融合为一维序列，使 Transformer 的自注意力机制能直接学习动作与视觉变化的对应关系，实现动作可控性。

- **稀疏关键帧记忆（Sparse Keyframe Memory）**
  - 在长时序建模中保存稀疏低分辨率关键帧（K=4），为生成过程提供全局时空约束；
  - 保持长时间一致性、防止累计漂移。

- **离散进度 token（Discrete Progress Token）**
  - 将任务完成度离散为多个阶段（0.0–1.0）；
  - 训练时借助视觉语言模型（SEED-1.5VL）打分；
  - 推理时联合预测未来图像与进度 token，实现无监督自动任务判定。

### 3. 学习与优化过程（文字化公式说明）

- 采用**掩码离散扩散（Masked Discrete Diffusion, MDD）**过程：
  - 从完整目标序列 \(y_{t+\Delta}\) 随机mask部分token；
  - 模型学习重建被mask部分，以最小化加权交叉熵损失；
  - 不同模态token设不同权重（进度token权重更高）。
- 推理阶段使用**并行去噪解码**迭代重建视觉帧与进度分数。

## 六、主要结论与发现

1. **动作可控性显著提升**：  
   - ∆LPIPS（专家/失败集合）均保持稳定（0.352），而基线方法在失败数据上急剧退化。
2. **长期一致性优越**：  
   - Round-trip LPIPS 在 H=20 时仅为 0.243，远低于基线（>0.45）。
3. **自动任务判定可靠**：  
   - 进度 token 输出与真实执行成功率的皮尔森相关系数 r≈0.91–0.93；
   - 可自动评分、无需人工判断或外部奖励函数。
4. **跨域泛化**：
   - 在仿真与现实中保持一致的 ∆LPIPS（0.31–0.36）；
   - 说明统一token化适应不同机器人形态与摄像视角。
5. **可用于真实策略排名**：  
   - 排名误差 MMRV=0.013，显著优于基线（最高至0.039）。

## 七、优点与创新点

- **架构创新**：首次将**离散扩散机制**用于机器人世界模型评估；
- **多模态统一建模**：视觉、语言、动作完全融合的token化序列；
- **自动评分机制**：不依赖VLM外部判别器，直接联合生成进度；
- **鲁棒控制依赖**：能真实反映“失败动作”，无“成功幻觉”；
- **可扩展性强**：统一生成机制适配不同机器人系统；
- **新指标提出**：∆LPIPS作为动态可控性量化标准，具解释力强。

## 八、不足与局限

- **计算资源要求高**：8×H800 GPU 对一般研究机构门槛较高；
- **任务覆盖性**：虽含三平台，但仍局限于桌面及双臂操作，未覆盖移动、柔性或长时协作任务；
- **依赖外部VLM标注**：训练阶段仍需用 SEED-1.5VL 生成离散进度标签；
- **物理精确性**：基于视觉生成的世界模型无法严格保真物理属性（如摩擦力、动力交互）；
- **实际部署评估差距**：尽管与真实表现相关性高，但仍存在建模偏差风险，特别在未见任务上。
