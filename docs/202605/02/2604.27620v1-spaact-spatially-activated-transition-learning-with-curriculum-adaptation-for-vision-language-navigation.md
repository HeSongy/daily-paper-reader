## 摘要
视觉-语言导航（Vision-and-Language Navigation，VLN）的目标是让具身智能体能够依据自然语言指令，在未知的三维环境中导航至目标位置。我们认为，将视觉语言模型（VLMs）适配于VLN任务，需要赋予模型两种互补能力以获取空间意识，即反向动作推理（原因，即why）和前向状态转移预测（方式，即how）。基于这一洞察，我们提出了SpaAct——一种简单而有效的训练框架，用以激活VLMs中的动态空间感知能力。具体而言，SpaAct引入了两项空间激活任务：动作回顾（Action Retrospection），促使模型根据视觉变化推断已执行的动作序列；以及未来帧选择（Future Frame Selection），要求模型基于历史和动作预测未来的视觉变化。这两个目标在反向动作推理和前向转移预测上提供了轻量监督，鼓励模型以符合VLM特性的方式构建动态空间意识。为进一步稳定适配过程，我们设计了TriPA——一种三因子渐进自适应课程学习方法（Tri-factor Progressive Adaptive curriculum learning），它通过从易到难组织训练样本，使模型能够逐步学习从基础移动到长程推理的导航技能。在标准VLN-CE基准上的实验结果表明，SpaAct能持续提升基于VLM的导航性能，并取得了当前最先进的表现。我们将公开代码与模型以支持未来研究。

## Abstract
Vision-and-Language Navigation (VLN) aims to enable an embodied agent to follow natural-language instructions and navigate to a target location in unseen 3D environments. We argue that adapting VLMs to VLN requires endowing them with two complementary capabilities for acquiring such awareness, namely backward action reasoning (why) and forward transition prediction~(how). Based on this insight, we propose SpaAct, a simple yet effective training framework that activates the dynamic spatial awareness in VLMs. Specifically, SpaAct introduces two spatial activation tasks: Action Retrospection, which asks the model to infer the executed action sequence from visual transitions, and Future Frame Selection, which forces the model to predict the visual transitions conditioned on history and action. These two objectives provide lightweight supervision on both backward action reasoning and forward transition prediction, encouraging the model to build dynamic spatial awareness in a VLM-friendly way. To further stabilize adaptation, we design TriPA, a Tri-factor Progressive Adaptive curriculum learning method that organizes training samples from easy to hard, allowing the model to gradually acquire navigation skills from basic locomotion to long-horizon reasoning. Experiments on standard VLN-CE benchmarks show that SpaAct consistently improves VLM-based navigation and achieves state-of-the-art performance. We will release the code and models to support future research.

## 一、核心问题与研究背景

- **任务定位**：视觉-语言导航（Vision-and-Language Navigation, VLN）要求智能体在未知三维环境中，根据自然语言指令完成目标导航任务。  
- **问题根源**：当前基于视觉语言模型（VLM）的VLN方法通常依赖语义理解（如识别目标物体或场景语义），但缺乏**动态空间意识**，即无法充分理解“视觉观察是如何随动作变化的”。  
- **研究动机**：  
  - 现有VLMs学到的主要是“语义认知”（what/where），而VLN需要“动作-空间关联”（why/how）。  
  - 无法建模动作引发的环境状态转变，使得导航策略常陷入**模式匹配式的表层学习**。  
- **核心目标**：赋予VLM同时具备“反向动作推理”（从状态变化推断动作因果）与“前向转移预测”（预测下一观察状态）的能力，从而弥合VLM的语义理解与空间认知之间的差距。

## 二、方法论：SpaAct 框架设计

### 2.1 核心思想
- **框架名称**：SpaAct（Spatially-Activated Transition Learning）。  
- **关键理念**：通过在VLM上引入两类**空间激活任务（Spatial Activation Tasks）**，显式鼓励模型建立动作条件下的动态空间感知。  
- **配合策略**：提出一种名为 **TriPA（三因素渐进自适应课程学习）** 的课程训练机制，从易到难地暴露样本，逐步稳定VLM的空间适配过程。

### 2.2 空间激活任务（Spatial Activation Tasks）
1. **Action Retrospection（AR，动作回溯）**  
   - **目标**：模型需根据相邻两帧视觉观测，预测导致这种视觉转变的动作序列（如"move_forward"、"turn_left"）。  
   - **核心思想**：将动作推理问题转化为语言生成问题，使空间一致性学习以VLM友好的方式融入训练。  
   - **损失函数**：使用标准交叉熵损失计算生成动作序列对的预测误差。  

2. **Future Frame Selection（FFS，未来帧选择）**  
   - **目标**：给定历史观测、当前帧与执行动作，要求模型从4个候选帧中选择正确的未来观测帧。  
   - **负样本策略**：
     - 同轨迹难负样本（相似但非真实帧）；
     - 跨轨迹随机负样本（增加多样性）。  
   - **形式化**：任务被构造为一个多项选择式视觉问答（VQA）问题；监督信号以分类交叉熵形式优化。

### 2.3 TriPA：三因素渐进自适应课程学习
- **目的**：稳定VLM适应过程，防止长程复杂样本早期训练造成梯度震荡。  
- **样本难度评分公式**：
  \[
  d_i = \alpha s_{traj,i} + \beta s_{instr,i} + \gamma s_{turn,i}
  \]
  其中：
  - \(s_{traj}\)：轨迹复杂度（路径长度对数归一化）；
  - \(s_{instr}\)：指令复杂度（词数）；
  - \(s_{turn}\)：运动复杂度（旋转动作比例）。  
- **阶段划分**：按难度分为四个 bucket（B0–B3），训练按比例从易到难逐步采样 （P1→P4），缓解“早期学习崩溃”。

### 2.4 整体训练流程
- **Stage 1：联合多任务训练**
  \[
  L_{stage1} = L_{nav} + \lambda_{ar}L_{ar} + \lambda_{ffs}L_{ffs}
  \]
  数据来源：  
  - 69 万对 AR 样本、69 万 FFS 样本、240 万导航样本（R2R-CE与RxR-CE）。  
- **Stage 2：辅助数据增强 + DAgger再训练**
  - 采样 11.7K DAgger 轨迹与 ~155K ScaleVLN 子集数据，优化主导航目标 \(L_{nav}\)。  
- 推理阶段仅保留导航支路，空间激活任务不带来额外推理成本。
