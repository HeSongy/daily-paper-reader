## 摘要
稳健的具身导航依赖于互为补充的感官线索。然而，在实践中，高质量且良好对齐的多模态数据往往难以获取。由于丰富的多模态输入会带来复杂的表征并显著扩大策略空间，训练一个单体模型也具有挑战性。轻量化的、专注于特定模态的智能体之间的跨模态协作提供了一种可扩展的范式。该方法在保留各模态优势的同时，实现了灵活部署与并行执行。本文提出了一个用于跨模态导航（Cross-Modal Navigation）的多智能体强化学习（Multi-Agent Reinforcement Learning, MARL）框架——CRONA（Cro-Na）。CRONA通过利用与控制相关的辅助信念以及具有全局状态的集中式多模态评论家（critic），提升了协作能力。在视觉-声学导航任务上的实验表明，多智能体方法在性能和效率上相比单智能体基线都有显著提升。我们发现，在显著线索下，具有有限模态的同质协作足以完成短程导航；而拥有互补模态的异质智能体协作通常更加高效；对于大规模复杂环境的导航，则需要更丰富的多模态感知和更高的模型容量。

## Abstract
Robust embodied navigation relies on complementary sensory cues. However, high-quality and well-aligned multi-modal data is often difficult to obtain in practice. Training a monolithic model is also challenging as rich multi-modal inputs induce complex representations and substantially enlarge the policy space. Cross-modal collaboration among lightweight modality-specialized agents offers a scalable paradigm. It enables flexible deployment and parallel execution, while preserving the strength of each modality. In this paper, we propose \textbf{CRONA}, a Multi-Agent Reinforcement Learning (MARL) framework for \textbf{Cro}ss-Modal \textbf{Na}vigation. CRONA improves collaboration by leveraging control-relevant auxiliary beliefs and a centralized multi-modal critic with global state. Experiments on visual-acoustic navigation tasks show that multi-agent methods significantly improve performance and efficiency over single-agent baselines. We find that homogeneous collaboration with limited modalities is sufficient for short-range navigation under salient cues; heterogeneous collaboration among agents with complementary modalities is generally efficient and effective; and navigation in large, complex environments requires both richer multi-modal perception and increased model capacity.

## 1. 核心问题与研究背景

### 研究动机
- **背景问题**：具身导航（Embodied Navigation）需要多种感知模态（如视觉、声学、深度、语言等）协同感知与决策。然而：
  - 高质量、对齐良好的多模态训练数据稀缺；
  - 单体（monolithic）多模态模型结构复杂，训练和部署成本极高；
  - 不同模态的噪声、时间结构、维度差异导致联合优化不稳定。
- **研究目标**：探索如何通过多智能体强化学习（MARL），实现**跨模态协作**（cross-modal collaboration），即让不同感官模态的智能体彼此配合完成导航任务，而无需在单体模型中对所有模态进行同步建模。

## 2. 方法论：CRONA 框架

### 核心思想
- 提出 **CRONA (Cross-Modal Navigation)** 框架，一种基于 **多智能体强化学习（MARL）** 的跨模态导航方法。
- 关键思路：将多模态导航任务拆分给不同的模态专属智能体（如视觉智能体、音频智能体），每个智能体仅感知其专长模态，并通过 **集中训练、分散执行**（CTDE）机制实现协作。

### 模型结构
CRONA 的整体架构包括：
1. **辅助信念预测器 (Auxiliary Belief Predictor)**  
   - 从复杂的声学输入（如双声道频谱谱图）中预测“控制相关的信念”（如目标位置、声源类别）。  
   - 信念更新采用指数滑动平均（EMA）平滑：  
     \[
     b_{i,t} = \alpha \hat{b}_{i,t} + (1-\alpha)b_{i,t-1}
     \]
   - 目的：缓解声学噪声和不确定性。

2. **基于注意力的历史编码器 (Attention-Based History Encoder)**  
   - 利用多头自注意力模块，对历史（观测–动作）序列进行编码；
   - 捕捉时间依赖与空间语义，为策略网络提供上下文嵌入。

3. **集中式多模态评论家 (Centralized Multi-Modal Critic)**  
   - 在训练时访问全局状态（如场景布局、目标位置），结合联合历史与信念向量评估全局价值；
   - 在执行时，所有智能体独立运行，仅根据各自局部历史执行动作；
   - 采用 PPO（Proximal Policy Optimization） + GAE（Generalized Advantage Estimation）优化策略与价值函数。

### 优化目标
整体损失由三部分组成：
- 策略梯度目标 \(J(\theta)\)：基于 PPO 的剪切策略目标；
- 价值网络损失 \(L(\phi)\)：基于剪切的值函数误差；
- 权重混合优化项：
  \[
  L = -\mu J(\theta) + (1-\mu)L(\phi)
  \]
其中共享编码器和变换器权重以实现高效联合训练。

## 3. 实验设计

### 数据与场景
- **Benchmark**：基于 **Matterport3D** 场景构建了新的跨模态协作导航基准。
- **五个代表性场景**：  
  1. **Studio** — 小型单房间（简单）  
  2. **Corridor** — 狭长走廊（声学提示主导）  
  3. **Apartment** — 多房型公寓（视觉主导）  
  4. **Ranch** — 中型多目标场景（跨模态协作）  
  5. **Maze** — 大型复杂迷宫（多模态主导）

每个场景包含不同的声源（如滴水、拍照、金属落地等），在 **Habitat** 模拟器与 **librosa** 音频环境下运行。

### 对比方法
1. **Single-Agent**：单体多模态模型（融合所有模态输入）。  
2. **VLA-Collab**（Vision-Language-Action）  
3. **ALA-Collab**（Audio-Language-Action）  
4. **AVLA-Collab**（全模态同质协作）  
5. **CRONA**（提出方法，异质跨模态协作）

### 评估指标
- 平均成功率（Succ）  
- 平均目标距离（Dist）  
- 目标检测率（Detect）  
- 平均步骤数与超时率（Steps / Timeout）

## 6. 主要结论与发现

- **总体性能**：CRONA 全局最优，平均成功率最高（95.72% 在 Studio 场景中）。
- **五种模态主导模式**：
  1. 无明显主导（No dominance）；
  2. 视觉主导（Apartment）；
  3. 听觉主导（Corridor）；
  4. 跨模态互补（Ranch）；
  5. 多模态融合主导（Maze）。
- **关键规律**：
  - 小场景中，同质协作已足够；
  - 异质跨模态协作在中等复杂环境最优；
  - 大场景需高模型容量 + 全模态感知。
- **辅助信念的重要性**：目标位置信念对于稳定训练与性能提升至关重要。
- **集中式评论家不可或缺**：移除全局状态输入几乎使模型失效。
