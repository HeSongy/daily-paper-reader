## 摘要
现有的视觉语言导航（Vision-Language Navigation，VLN）方法通常采用以自我为中心的逐步导航范式，这种方式容易导致误差累积并限制导航效率。尽管近期的一些方法尝试利用预构建的环境地图，但它们通常依赖于逐步更新的记忆图或对离散路径方案进行评分，这限制了连续的空间推理并造成离散化瓶颈。我们提出了自上而下视觉语言导航（Top-Down VLN，TD-VLN），将导航重新定义为在预构建的自上而下地图上进行的一步全局路径规划问题，并基于我们新构建的 R2R-TopDown 数据集实现这一任务。为解决该问题，我们引入了 NavOne——一个统一的框架，可以在单次端到端前向传递中直接预测多模态地图上的密集路径概率。NavOne 包含一个自上而下地图融合器（Top-Down Map Fuser），用于联合的多模态地图表示，并扩展了注意力残差机制，以实现具备空间感知的深度混合。大量基于 R2R-TopDown 的实验表明，NavOne 在基于地图的 VLN 方法中达到了当前最先进的性能，在规划阶段比现有基于地图的基线方法提速 8 倍，比以自我为中心的方法提速 80 倍，从而实现了高效的全局导航。

## Abstract
Existing Vision-Language Navigation (VLN) methods typically adopt an egocentric, step-by-step paradigm, which struggles with error accumulation and limits efficiency. While recent approaches attempt to leverage pre-built environment maps, they often rely on incrementally updating memory graphs or scoring discrete path proposals, which restricts continuous spatial reasoning and creates discrete bottlenecks. We propose Top-Down VLN (TD-VLN), reformulating navigation as a one-step global path planning problem on pre-built top-down maps, supported by our newly constructed R2R-TopDown dataset. To solve this, we introduce NavOne, a unified framework that directly predicts dense path probabilities over multi-modal maps in a single end-to-end forward pass. NavOne features a Top-Down Map Fuser for joint multi-modal map representation, and extends Attention Residuals for spatial-aware depth mixing. Extensive experiments on R2R-TopDown show that NavOne achieves state-of-the-art performance among map-based VLN methods, with a planning-stage speedup of 8x over existing map-based baselines and 80x over egocentric methods, enabling highly efficient global navigation.

  - 将导航任务从**逐步动作预测**重新定义为**在自上而下地图（top-down map）上的一步全局路径规划**问题。

## 二、方法论：NavOne 框架

### 1. 核心思想
- 将 VLN 视为**全局路径预测问题**：  
  给定语言指令与多模态俯视地图（RGB、占据、语义），直接预测从起点到目标的连续路径。  
- 模型一次前向传播完成整条路径的推理，无需逐步动作生成。

### 2. 架构总览
NavOne 包含三个主要组件：

#### (1) Top-Down Map Fuser（地图融合器）
- 输入三种地图：
  - **RGB map (3 通道)**：环境外观；
  - **Occupancy map (1 通道)**：可通行区域；
  - **Semantic map (41 类语义嵌入)**：环境类别与物体信息。
- 通过学习的语义嵌入层与通道拼接，形成统一的多模态张量。  

#### (2) Path Former（编码器–解码器主干）
- **输入：** 地图张量 + 语言指令 + 初始姿态。  
- **语言编码：** 使用冻结的 **BERT-base**。  
- **多模态融合：**  
  - 位置信息与旋转信息投射为与视觉特征同维向量后相加（element-wise addition）。  
  - 采用多层**跨模态注意力（Cross-Attention）**，使地图特征逐层获取与任务相关的语言语义。  
- **视觉骨干：** Vision Transformer (ViT-B/16)，引入**扩展注意力残差 Attention Residuals (AR)** 。  
  - **创新点**：提出**Spatial-Aware Depth Query**（空间感知深度查询），使不同位置可独立调节层间混合权重，从而增强空间推理能力。  
- **解码部分：**  
  - 逐层反卷积上采样（transposed conv），生成与输入地图同分辨率的两张概率图：  
    - **路径概率图** (P_path)：每个像素为路径概率；  
    - **目标概率图** (P_goal)：预测目标分布。

#### (3) Path Extractor（路径提取器）
- 采用改进的 **A\*** 搜索，在路径概率图与占据图上搜索最优可行路径：
  - 成本函数：  
    - 可行区成本 = \(1 / (P_{path} + ε)\)  
    - 障碍区成本 = 大惩罚常数 \(C_{obs}\)  
  - 以目标概率最大处为目标点，根据欧氏启发式找到最短可执行路径。  
- 失败时执行“最近可达点回退”机制，保证路径可执行性。

### 3. 训练目标
- 采用联合多任务损失：
  - 路径预测、目标预测分别使用 BCE 损失；
  - 引入**路径连续性正则项** \(L_{continuity}\)：  
    - 包含梯度平滑、起点约束、形态腐蚀一致性三部分。  
  - 总损失：  
    \[
    L_{total} = L_{path} + αL_{goal} + λL_{continuity}
    \]
- 典型参数：α=5.0，λ=0.1。

## 七、优点与创新亮点

- **任务重构创新：**  
  - 首次将 VLN 形式化为在俯视图上的“一步全局路径预测”，摆脱时间序列决策约束。

- **数据贡献：**
  - 提供全新多模态数据集 R2R-TopDown（RGB + occupancy + semantic）。  
  - 支持研究从语言到路径的直接映射任务。
- **系统优势：**
  - 推理一次完成整条路线，极大减少计算成本；
  - 输出具备空间可视解释（路径与目标地图）。
