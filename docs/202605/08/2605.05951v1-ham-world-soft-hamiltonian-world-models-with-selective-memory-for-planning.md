## 摘要
世界模型通过学习的潜在动态实现基于模型的规划，但随着规划范围的延长或动态分布的变化，想象的滚动预测会变得不稳定。我们认为这种不稳定性反映了规划器所使用的潜在空间中缺失的两种结构：一种是基于历史的记忆，用于实现近似的马尔可夫完备性；另一种是几何组织，用于区分配置、动量和任务语义。我们提出 HaM-World（HMW），一种结构化的世界模型，它将潜在状态分解为规范的 (q, p) 子空间和上下文子空间 c，同时利用 Mamba 选择性状态空间记忆作为同一潜在动态的历史条件输入。在此框架中，(q, p) 通过由能量导出的哈密顿向量场以及可学习的残差/控制动态进行演化，而 c 则捕获语义、耗散及非保守因素。这使规划器能够共享一个潜在状态，用于动态预测、奖励/价值估计、模拟滚动以及 CEM 动作搜索。在四个 DeepMind Control Suite 任务上，HaM-World 达到最高平均 AUC（117.9，+9.5%），将长范围滚动误差降低至强基线模型的 45%，并在 {3,5,7} MSE 测试单元中赢得 11/12 项。在 12 种跨动态变化、动作延迟与观测遮蔽的 OOD 扰动下，HaM-World 在每种条件中均实现最高收益，在 Finger Spin 任务上的平均 OOD 收益提升为 10.2%，在 Reacher Easy 上提升为 13.6%。机制诊断进一步表明，该模型具有有界的无动作哈密顿能漂移、策略滚动下的结构化能变化，以及由控制引起的连贯能量传递，有力地支持了 Soft-Hamiltonian 动力学设计的预期效果。

## Abstract
World models enable model-based planning through learned latent dynamics, but imagined rollouts become unstable as the planning horizon grows or the dynamics distribution shifts. We argue that this instability reflects two missing structures in planner-facing latents: history-conditioned memory for approximate Markov completeness, and geometric organization that separates configuration, momentum, and task semantics. We propose HaM-World (HMW), a structured world model that decomposes the latent state into a canonical (q, p) subspace and a context subspace c, while using Mamba selective state-space memory as the history-conditioned input to the same latent dynamics. Within this interface, (q, p) evolves through an energy-derived Hamiltonian vector field plus learnable residual/control dynamics, while c captures semantic, dissipative, and non-conservative factors. This gives the planner a single latent state shared by dynamics prediction, reward/value estimation, imagined rollouts, and CEM action search. On four DeepMind Control Suite tasks, HaM-World reaches the highest Avg. AUC (117.9, +9.5%), reduces long-horizon rollout error to 45% of a strong baseline model, and wins 11/12 k in {3,5,7} MSE cells. Under 12 OOD perturbations spanning dynamics shifts, action delay, and observation masking, HaM-World achieves the highest return in every condition, with average OOD-return gains of 10.2% on Finger Spin and 13.6% on Reacher Easy. Mechanism diagnostics further show bounded action-free Hamiltonian-energy drift, structured energy variation under policy rollouts, and coherent control-induced energy transfer, supporting the intended Soft-Hamiltonian dynamics design.

## 一、研究动机与核心问题

- **背景**：世界模型（World Models）在模型驱动强化学习中用于表示环境的潜在动态结构，可被规划器反复调用以执行“想象式”滚动（imagined rollouts）。然而，随着规划范围增长或环境分布发生漂移（例如动力学变化、动作延迟），这些滚动结果往往会变得不稳定。  
- **主要问题**：
  - 现有模型主要基于相关性拟合而非物理结构，缺乏**结构化归纳偏置**；
  - 潜在状态通常混杂了配置、动量、语义信息，导致**多步预测误差急剧累积**；
  - 在部分可观测场景下，模型的状态更新缺乏可追溯的历史记忆，使得**马尔可夫完备性难以保证**。
- **研究目标**：建立一种既能捕获物理结构（几何一致性），又能利用历史条件保证 Markov 性的世界模型，从而在长时规划中保持稳定与泛化能力。

## 二、方法论：HaM-World 模型设计

### 1. 核心思想
HaM-World（HaM = **Soft-Hamiltonian + Mamba Memory**）提出**“记忆–几何双结构”世界模型**：
- **输入侧**：通过 *Mamba* 选择性状态空间模型（SSM），引入历史依赖，使潜在动态具备近似马尔可夫完备性；
- **输出侧**：将潜在空间拆解为 `(q, p, c)` 三部分：
  - `(q, p)`：构成 *Soft-Hamiltonian* 动力子空间；
  - `c`：承载任务语义、耗散与非保守因素。

### 2. 动力学机制
- **Soft-Hamiltonian 核心方程**：
  - 基于能量函数 \( H_\phi(q, p) \)，定义混合更新：
    \[
    \begin{aligned}
    \Delta q_t &= (1-α)\Delta q^{net}_t + α∂_p H_t, \\
    \Delta p_t &= (1-α)\Delta p^{net}_t - α∂_q H_t + G_t a_t,
    \end{aligned}
    \]
  其中：
  - \(α\) 为可调混合系数；
  - \(G_t a_t\) 表示验证控制驱动；
  - “Soft” 表示在能量守恒与数据拟合间的柔性平衡。
- **Mamba 记忆机制**：
  - 历史状态 \(h_t = \text{Memory}_\phi(z_t, a_t, h_{t-1})\)；
  - 输出作为同一潜在动态的条件输入，而非独立的并行模型；
  - 解决了部分可观测、延迟控制等非马尔可夫问题。

### 3. 损失与训练目标
总体损失：
\[
L_{total} = L_{repr} + β_{dyn}L_{dyn} + β_{roll}L_{roll} + β_rL_{reward} + β_vL_{value} + β_pL_{policy} + β_{geo}L_{geo}
\]
其中关键项包括：
- **表示与一致性损失**：类似 DreamerV3 的表示与一/多步预测；
- **几何正则项**：Hamiltonian 对齐 (L_ham)、能量漂移惩罚 (L_energy)、小动作平滑 (L_sa)、子空间解耦 (L_dec) 等；
- **规划器**：基于 **CEM（Cross-Entropy Method）** 优化，直接在潜在空间中搜索动作序列，无需独立 actor。

### 4. 算法流程
1. **编码器**：从观测得到潜在状态 `[q, p, c]`；
2. **记忆更新**：通过 Mamba SSM 产生历史状态 `h_t`；
3. **动态预测**：执行 Soft-Hamiltonian 更新；
4. **奖励与价值评估**：用预测状态估计；
5. **CEM 规划**：基于模型 rollouts 迭代优化动作序列；
6. **统一接口训练**：规划、预测、奖励、价值均共享同一潜在动力学。
