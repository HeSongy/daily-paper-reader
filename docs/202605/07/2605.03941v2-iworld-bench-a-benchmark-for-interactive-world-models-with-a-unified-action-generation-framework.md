## 摘要
实现人工通用智能（AGI）需要具备自适应学习与交互能力的智能体，其中交互式世界模型为感知、推理与行动提供了可扩展的环境。然而，目前的研究仍然缺乏用于评估其物理交互能力的大规模数据集和统一基准。为此，我们提出了 iWorld-Bench——一个全面的基准，用于在交互相关能力（如距离感知与记忆）上训练和测试世界模型。我们构建了一个包含 33 万段视频片段的多样化数据集，并精选了 2100 个高质量样本，涵盖多种视角、天气和场景。由于现有世界模型在交互模态上存在差异，我们提出了一个动作生成框架（Action Generation Framework）来统一评估，并设计了六种任务类型，生成了 4900 个测试样本。这些任务共同评估模型在视觉生成、轨迹跟随和记忆等方面的性能。我们对 14 个具有代表性的世界模型进行了评估，识别了关键局限性，并为未来研究提供了启示。iWorld-Bench 模型排行榜已在 iWorld-Bench.com 上公开。

## Abstract
Achieving Artificial General Intelligence (AGI) requires agents that learn and interact adaptively, with interactive world models providing scalable environments for perception, reasoning, and action. Yet current research still lacks large-scale datasets and unified benchmarks to evaluate their physical interaction capabilities. To address this, we propose iWorld-Bench, a comprehensive benchmark for training and testing world models on interaction-related abilities such as distance perception and memory. We construct a diverse dataset with 330k video clips and select 2.1k high-quality samples covering varied perspectives, weather, and scenes. As existing world models differ in interaction modalities, we introduce an Action Generation Framework to unify evaluation and design six task types, generating 4.9k test samples. These tasks jointly assess model performance across visual generation, trajectory following, and memory. Evaluating 14 representative world models, we identify key limitations and provide insights for future research. The iWorld-Bench model leaderboard is publicly available at iWorld-Bench.com.

## 二、方法论与技术框架

### 1. 整体思路

- 构建一个含 **33 万视频片段的多模态数据集**；
- 提出一个**动作生成框架（Action Generation Framework, AGF）**，统一各种交互模态；
- 设计 **6 类任务** 全面评估模型的「视觉生成、动作跟随、记忆保持」三大维度；
- 定义 **9 项量化指标**，从图像质量、轨迹精确度、记忆一致性等方面刻画性能。

### 2. 动作生成框架（AGF）细节

- **核心结构**：  
  将交互动作统一表示为四元组 \(C_t = [D, T, R, V]\)，其中：
  - \(D\)：动作难度等级（1–6）
  - \(T\)：平移动作ID（27种基本动作）
  - \(R\)：旋转动作ID（27种）
  - \(V\)：有效性标志（常见/罕见动作）
- **统一编码映射**：
  - 将不同模型输入模态（如文本命令、键盘信号、相机外参）统一映射到一个**81维标准动作空间**；
  - 这一编码机制实现了模态无关（modality-agnostic）的控制对齐。

### 3. 任务类型设计（共6类）

| 类型 | 说明 | 样本数量 |
| ---- | ---- | ---- |
| Action Control D1–D4 | 不同难度等级的动作控制（单轴到多轴移动） | 各1000条任务 |
| Memory Ability | 循环路径任务，用于测试记忆一致性 | 200条任务 |
| Camera Following | 基于相机参数的路径跟随任务 | 700条任务 |

### 4. 评测指标体系（共9项）

- **视觉质量指标**（4项）：图像清晰度、亮度一致性、色温一致性、纹理锐度；
- **动作执行指标**（3项）：轨迹平滑性、控制精度、执行容差；

公式基于余弦相似度、多尺度质量评价（MUSIQ）、LPIPS-SSIM-MSE混合指标的加权组合。

## 六、主要发现与结论

1. **HY-World 1.5** 在动作可控性和记忆保持方面表现最佳（平均得分 0.7873）。  
2. 文本控制模型（如 CogVideoX-I2V）虽然生成质量最高，但动作追踪能力较弱。  
3. 相机参数控制模型（如 AC3D、VideoX-Fun-Wan）在轨迹容差与记忆任务上表现出最强稳定性。  
4. 存在普遍现象：**视觉质量与控制精度呈负相关**，两者存在内在取舍。  
5. iWorld-Bench 的自动评分结果与人工评估的Spearman相关系数达 **0.8053 (p < 0.001)**，验证其一致性与可信度。  
6. 基准揭示了当前交互式世界模型尚未能兼备高保真视觉与高可控动态响应。
