## 摘要
时间对比表征学习能够诱导一种潜在结构，使得长时间规划可以在低维线性系统中转化为推理问题。然而，现有的对比规划工作只学习到单一的潜在几何结构，无法区分在相同起点-目标查询下，为任务效率与风险暴露之间权衡的多种合理行为。我们提出了 MoMo，一种基于偏好条件的对比规划器，它允许通过一个标量用户偏好在推理时连续调节规划的保守程度，而无需重新训练。MoMo 分别通过特征层面的线性调制（Feature-Wise Linear Modulation）和低秩神经调制（low-rank neural modulation）学习表征几何与潜在预测算子的联合条件结构。我们展示了该方法保持了表征空间中用于基于推理的对比规划的概率密度比，同时保留了推理阶段的高效性。在六个环境中，MoMo 能够根据用户偏好平滑地调整规划安全性，相较于状态增强基线方法展现出更优的时间一致性和偏好一致性。

## Abstract
Temporally contrastive representation learning induces a latent structure capable of reducing long-horizon planning to inference in a low-dimensional linear system. However, existing contrastive planning work learns a single latent geometry which cannot distinguish multiple valid behaviors trading task efficiency against risk exposure for the same start-goal query. We introduce MoMo, a preference-conditioned contrastive planner allowing a scalar user preference to continuously modulate plan conservativeness at inference time, without retraining. MoMo learns a joint conditioning of the representation geometry and latent prediction operator via Feature-Wise Linear Modulation and low-rank neural modulation, respectively. We show that our formulation preserves the probability density ratio encoded in the representation space that is required for inference-driven contrastive planning, further retaining its inference-time efficiency. Across six environments, MoMo smoothly adapts plan safety according to user preferences, yielding improved temporal and preferential consistency over state augmentation baselines.

## 一、研究背景与核心问题

- **研究背景**  
  传统的**对比式表征学习（Contrastive Representation Learning）**已在规划与控制任务中展现出较强能力。尤其是时间对比（Temporally Contrastive）方法能够学习到**潜在时间可达性几何结构**，将长时间规划问题转化为低维线性系统推理，从而大幅提升规划效率。

- **核心问题**  
  现有的对比规划方法仅学习**单一的潜在几何结构（latent geometry）**，不能区分在同一任务条件下不同**风险—效率权衡**的多种可行行为。例如：
  - 在同一“起点–目标”条件下，某些轨迹可能更安全但冗长，另一些则更高效但高风险。
  - 当前模型无法根据用户偏好灵活选择这种平衡。

- **研究动机**  
  作者希望让规划器在无需重新训练的情况下，**根据用户在推理时提供的安全偏好（scalar preference）**平滑调节行为保守度，实现**偏好条件（preference-conditioned）的规划**，增强机器人在未知动态环境中执行任务时的自适应性与安全性。

## 二、方法论：MoMo 框架与核心技术

### 2.1 总体思路
MoMo（**Modulated Motion Planner**）是在 CRLP（Contrastive Representation Learning for Planning）框架基础上的扩展。其目标是：  
> 在保持对比规划可推理结构的同时，引入一个可连续调节的用户偏好变量 Δ，以在推理阶段实时修改规划风格。

MoMo 通过**联合调节（joint conditioning）**实现：
- 对表征空间几何结构的调制（representation geometry）；
- 对潜在状态转移算子（latent prediction operator）的调制。

### 2.2 核心组件

1. **偏好统计量（Preference Estimation）**  
   - 从轨迹的风险代价序列中提取**CVaR（条件风险价值）**为偏好标签：  
     \[
     \tilde{\Delta}_i = \text{CVaR}_\alpha (\tau_i)
     \]
   - 将结果归一化至 [0,1]，其中 0 表示最安全、1 表示最激进。

2. **偏好调制的表征几何 (Preference-Conditioned Encoder)**  
   - 通过 **FiLM（Feature-wise Linear Modulation）** 层在编码器各层注入偏好信号：  
     \[
     a^{(l)}_\Delta = \text{swish}\big(\lambda^{(l)}_\Delta \odot h^{(l)} + \beta^{(l)}_\Delta\big)
     \]
   - 其中 λ 与 β 由偏好输入 Δ 通过 MLP 生成，允许对隐藏特征通道进行层级性调幅。

3. **偏好调制的预测算子 (Preference-Conditioned Latent Dynamics)**  
   - 使用**低秩分解**对预测矩阵 A 进行偏好依赖性建模：
     \[
     A_\Delta = A_0 + U(\Delta)V(\Delta)^\top
     \]
   - 其中 A₀ 是基础动力学项，U 和 V 是低秩调制网络（rank < dz）。
   - 保证各偏好下主要动力学一致，仅在局部方向上调整过渡结构。

4. **对比学习目标（Conditioned InfoNCE Loss）**  
   - 扩展 Eysenbach 等人的对比目标，引入偏好相关的正负样本：
     \[
     L_{\text{MoMo}} = -\sum_i \Big[ 
     \log\frac{S_{ii}^{\Delta_i}}{\sum_j S_{ij}^{\Delta_i}} +
     \log\frac{S_{ii}^{\Delta_i}}{\sum_j S_{ji}^{\Delta_j}}
     \Big]
     \]
   - 证明在最优条件下保持密度比性质：
     \[
     S^{\Delta*}_{ij} \propto \frac{p_\gamma(s_j^+|s_i, \Delta)}{p(s_j^+)}
     \]
   - 保留高效的**高斯–马尔可夫（Gauss–Markov）推理结构**。

5. **规划与推理流程（Algorithm 1）**
   - **训练阶段**：基于带偏好标签的轨迹对进行对比学习。
   - **推理阶段**：
     1. 计算端点的编码向量；
     2. 构造偏好调制的 AΔ；
     3. 在潜空间中解高斯马尔可夫后验得到 z₁:n；
     4. 通过最近邻解码回观测空间获得实际路径。

## 八、不足与局限

1. **计算复杂度**：  
   训练阶段从 O(B) 升至 O(B²)，因偏好嵌入需多组对比采样。
2. **推理依赖覆盖性**：  
   需要在 held-out 数据中有充分偏好覆盖，否则最近邻解码可能产生不安全的状态。
3. **假设限制**：  
   承继 CRLP 假设（密度比编码与高斯性分布），理论上可能受逼近误差影响。
4. **数据限制**：  
   仅使用离线数据，无法自主探索新的偏好—状态组合。
5. **偏好单维性**：  
   使用单一标量 Δ 表征风险偏好，无法表达多维偏好如速度、能耗等复合权衡。
