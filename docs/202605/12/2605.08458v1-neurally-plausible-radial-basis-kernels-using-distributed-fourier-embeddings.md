## 摘要
连贯、连续的空间表示对于将物理与感知现象综合到同一个表示空间中至关重要。径向基核函数为实现这类分布式表示提供了一条可行途径。在本研究中，我们旨在刻画并分析在空间语义指针（spatial semantic pointers）这一神经可行框架下可实现的常见径向基核函数。此外，我们还分析了基于类似网格细胞（grid cell-like）表示的先前径向基核函数研究，并证明这类表示不仅能够实现径向基核函数，而且在实现该类核函数方面具有最优性。

## Abstract
Coherent, continuous spatial representations are critical for synthesizing physical and perceptual phenomena into a single representational space. Radial basis kernels provide a path forward for this type of distributed representation. In this work, we aim to characterize and analyze common radial basis kernels realizable in the neurally-plausible framework of spatial semantic pointers. Further, we analyze previous radial basis kernel work based on grid cell-like representations and demonstrate that such representations are both capable of and optimal for realizing radial basis kernels.

## 二、方法论与理论框架

### 2.1 核心思想

- 基于 **Bochner 定理**，利用傅里叶变换的概率分布与平移不变核之间的对应关系，实现径向基核构造：
  \[
  K(\mathbf{x}) = \int e^{j\boldsymbol{\omega}^\top \mathbf{x}} p(\boldsymbol{\omega}) d\boldsymbol{\omega}
  \]
  其中 \(p(\boldsymbol{\omega})\) 为相位分布。  
- 通过假定相位分布为中心对称且方向均匀（即 **各向同性分布**），推导出广义的 n 维径向基核形式。
- 将相位向量的方向与幅度解耦后得到一般径向基公式（由贝塞尔函数表征）：
  \[
  K(\mathbf{x}) = \Gamma\left(\frac{n}{2}\right) \int_0^\infty p(r) \left(\frac{2 r \|\mathbf{x}\|_2}{2}\right)^{\frac{n}{2}-1} J_{\frac{n}{2}-1}(r \|\mathbf{x}\|_2) dr
  \]
  （论文中记为 Equation (1)）

### 2.2 三种主要核形式

1. **超几何（Hypergeometric）核**：  
   - 从均匀分布 \(r \sim U(0,\lambda)\) 推导。  
   - 结果形式可展开为超几何函数或其级数展开。  
   - 对应 HexSSP（六边形空间语义指针）的典型核。  

2. **高斯（Gaussian）核**：  
   - 从卡方分布 \(r \sim \chi(n)\) 推导，可得  
     \[
     K(\mathbf{x}) = e^{-\|\mathbf{x}\|^2}
     \]
     （Equation 2）。  
   - 维度变化不影响核形状。  

3. **Jinc 核（n-jinc）**：  
   - 从 Beta 分布 \(r \sim \beta_\lambda(n,1)\) 构造。  
   - 包含贝塞尔函数项，二维情况下退化为 jinc 函数形式：
     \[
     K(\mathbf{x}) = \frac{J_0(\ell^{-1}\|\mathbf{x}\|)}{\|\mathbf{x}\|}
     \]
     （Equation 5）。  

### 2.3 网格细胞表示建模

- 将网格细胞建模为多个旋转的 n 维单纯形（simplex）相位矩阵。  
  - 旋转由 **SO(n)** 群的 Haar 分布采样实现。  
  - 每个网格单元的取样方向均匀分布于单位球面。  
- 多尺度与多方向组合后，整体相位分布近似 **各向同性分布**。  
- 证明网格细胞样结构在采样效率与空间覆盖上为最优配置，可近似任何径向基核。
