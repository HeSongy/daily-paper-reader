## 摘要
我们提出了 Action Agent，这是一种双阶段框架，将智能导航视频生成与受流约束的扩散控制相结合，用于多体机器人导航。在第一阶段（Stage I），大型语言模型（LLM）充当编排模块，选择视频扩散模型，通过迭代验证优化提示词，并积累跨任务记忆，以从语言和图像输入中合成物理上合理的第一人称导航视频。该方法使视频生成成功率在 50 个导航任务中由单次生成的 35% 提高到 86%。在第二阶段（Stage II），我们引入了 FlowDiT，一种受流约束的扩散 Transformer，可通过动作空间去噪扩散将优化后的目标视频和语言指令转化为连续的速度指令。FlowDiT 集成了 DINOv2 视觉特征、学习到的光流进行自运动表示，以及用于语义停靠的 CLIP 语言嵌入。我们首先在 RECON 户外导航数据集上进行预训练，并在 Isaac Sim 中收集的 203 个 Unitree G1 仿人机器人片段上进行微调，以校准速度动态。单个 4300 万参数的模型在仿真中获得 73.2% 的导航成功率，并在真实 Unitree G1 上，在未见的室内环境下以开环执行完成 64.7% 的任务，同时以 40–47 Hz 的频率运行。我们在三种具身形态上评估了 Action Agent：一个实际的 Unitree G1 仿人机器人、一架无人机和一个轮式移动机器人（Isaac Sim），结果表明，将轨迹想象与执行解耦可形成一种可扩展且具身感知的语言引导导航范式。

## Abstract
We present Action Agent, a two-stage framework that unifies agentic navigation video generation with flow-constrained diffusion control for multi-embodiment robot navigation. In Stage I, a large language model (LLM) acts as an orchestration module that selects video diffusion models, refines prompts through iterative validation, and accumulates cross-task memory to synthesize physically plausible first-person navigation videos from language and image inputs. This increases video generation success from 35% (single-shot) to 86% across 50 navigation tasks. In Stage II, we introduce FlowDiT, a Flow-Constrained Diffusion Transformer that converts optimized goal videos and language instructions into continuous velocity commands using action-space denoising diffusion. FlowDiT integrates DINOv2 visual features, learned optical flow for ego-motion representation, and CLIP language embeddings for semantic stopping. We pretrain on the RECON outdoor navigation dataset and fine-tune on 203 Unitree G1 humanoid episodes collected in Isaac Sim to calibrate velocity dynamics. A single 43M-parameter checkpoint achieves 73.2% navigation success in simulation and 64.7% task completion on a real Unitree G1 in unseen indoor environments under open-loop execution, while operating at 40--47 Hz. We evaluate Action Agent across three embodiments: a Unitree G1 humanoid (real hardware), a drone, and a wheeled mobile robot (Isaac Sim), demonstrating that decoupling trajectory imagination from execution yields a scalable and embodiment-aware paradigm for language-guided navigation.

## 一、研究动机与核心问题

- **背景**：语言条件下的机器人导航是具身智能（Embodied AI）的核心挑战。尤其在多形态机器人（仿人、轮式、无人机）场景中，如何将自然语言的高层意图转换为物理可执行的低层运动指令，长期存在两方面困难：
  1. 视频生成阶段缺乏物理合理性，常出现“穿墙、悬浮”等幻象。
  2. 动作控制阶段难以保持连续、稳定的速度与空间一致性。

- **传统方法局限性**：
  - **模块化导航管线（SLAM+规划+控制）**：可解释性强但难以泛化，需依机器人和传感器细调。
  - **端到端视觉-语言-动作模型（VLA）**：计算成本高（参数量动辄数十亿），并且将意图与控制强耦合，缺乏物理解释能力。

- **核心问题**：如何形成一种**可解释、跨具身体、物理合理且高效的语言导航机制**，既能利用生成模型的视觉理解能力，又能保持物理一致性与实时性。

## 二、方法论与技术框架

### 2.1 整体思路：两阶段解耦架构
论文提出 **Action Agent** —— 一个将 “轨迹想象” 与 “动作执行” 明确分离的双阶段系统：

1. **Stage I — Agentic Video Generation（数字排练）**
   - 形式化为：  
     \( (L, I_0) \rightarrow V_{goal} \)，其中  
     - \( L \)：语言指令  
     - \( I_0 \)：初始场景观察图像  
     - \( V_{goal} \)：验证过的目标导航视频（Visual Intermediate Representation, VIR）
   - **核心机制**：一个大型语言模型（LLM）充当“编排智能体”，通过迭代式提示优化生成物理合理的导航视频。
   - **算法流程（文字版）**：
     1. LLM 读取任务与图像，依据长短期记忆选择视频生成模型（如 WAN 2.2 或 LTX-Video）。
     2. 调用视觉-语言模型构造增强提示。
     3. 生成视频后由推理评估模块（Cosmos-Reason1）评分三项指标：  
        - Prompt Adherence — 与指令一致度  
        - Physical Plausibility — 物理合理性  
        - Visual Quality — 视觉质量
     4. 若平均分低于阈值 80，则识别瓶颈并局部改进提示词。
     5. 循环迭代至收敛或迭代上限 \(K_{max}=5\)。

   - **三层记忆架构**：  
     - **规则库（rules.md）**：固定优化约束与格式。  
     - **长期记忆（MLT）**：跨任务保存成功策略与模板。  
     - **短期记忆（MST）**：防止循环失败策略。

2. **Stage II — FlowDiT（流约束扩散变换器）**
   - 形式化为：  
     \( (V_{goal}, [I_t], L) \rightarrow a_t \)，其中  
     - \( I_t \)：实时观察图像（可选）  
     - \( a_t \)：机器人连续速度指令。
   - **核心结构**：Diffusion Transformer（DiT）架构的动作扩散模型，通过视觉、流、语言多模态条件生成动作序列。
   - **模型要点**：
     - **输入融合向量（2304维）**，包含：
       - DINOv2视觉特征（场景结构）
       - 光流CNN表示（局部自运动）
       - CLIP语言嵌入（语义停靠）
       - 当前观察视觉特征（可选）
     - **训练目标**：预测噪声以重构干净动作序列（8步时间窗口，三维速度）。
     - **采样方式**：DDIM推理，10步去噪；执行首步动作并循环预测（receding horizon control）。
     - **时序条件注入机制**：采用 adaLN-Zero 模块化层归一调制，实现轻量高效的条件控制。
