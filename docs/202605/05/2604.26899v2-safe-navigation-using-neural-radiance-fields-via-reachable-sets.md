## 摘要
在复杂环境中实现安全导航是自主系统面临的重要挑战。机器人在充满障碍的场景中进行导航时，必须能够在存在多种几何形态的障碍物、目标和自体对象情况下安全地移动。在本研究中，利用机器人的状态空间中可达集的表示来刻画其实时能力，以满足安全导航的需求。与此同时，神经辐射场（Neural Radiance Fields, NeRFs）被用于根据需要计算、存储和操作障碍物或自体车辆的体积表示。受限最优控制被用来构建由线性矩阵不等式约束组成的路径规划问题。我们在两个不同场景下针对大量障碍物进行了路径规划的仿真实验。通过在相应的受限最优控制问题中使用可达集，验证了安全导航的效果。

## Abstract
Safe navigation in cluttered environments is an important challenge for autonomous systems. Robots navigating through obstacle ridden scenarios need to be able to navigate safely in the presence of obstacles, goals, and ego objects of varying geometries. In this work, reachable set representations of the robot's real-time capabilities in the state space can be utilized to capture safe navigation requirements. While neural radiance fields (NeRFs) are utilized to compute, store, and manipulate the volumetric representations of the obstacles, or ego vehicle, as needed. Constrained optimal control is employed to represent the resulting path planning problem, involving linear matrix inequality constraints. We present simulation results for path planning in the presence of numerous obstacles in two different scenarios. Safe navigation is demonstrated through using reachable sets in the corresponding constrained optimal control problems.

## 二、方法论：NeRF 可达集联合导航模型

### 1. 系统建模
- 机器人状态方程为线性时变系统：
  \[
  \dot{x}(t) = A(t)x(t) + B(t)u(t)
  \]
  其中 \(x \in \mathbb{R}^n\)、\(u \in \mathbb{R}^m\)、初始状态集合为 \(X_0\)。

- 目标：在有限时间 \(T\) 内找到控制输入 \(u(t)\)，使得机器人从起点移动至目标集 \(G\)，同时满足避障约束 \(S(τ)∩O_i = ∅\)。

### 2. 可达集的核心思想
- **定义**：给定动态系统和控制范围，所有可能到达的状态点构成的集合 \(R(T; X_0, U)\)。  
- **性质**：在**线性系统和多面体输入约束**下，可达集仍为**多面体**。  
- **计算**：采用 Varaiya (2000) 的极点传播原理，将初始集边界的接触点通过极点最优控制传播得到未来时刻的可达集：
  - 更新点的运动：\(\dot{z}_j(τ) = A(τ)z_j(τ) + B(τ)u_j^*(τ)\)
  - 控制律：\(u_j^*(τ) = \arg\max_{u∈U}\ ⟨c_j(τ), A(τ)z_j(τ) + B(τ)u⟩\)

### 3. NeRF 体积建模
- 使用神经辐射场学习障碍物、机器人或目标的三维形状与外观：
  - 输入为不同视角的图像序列 \(I_i(x, y, z, θ, ϕ)\)；
  - 输出为体积辐射函数 \(F: \mathbb{R}^3\times \mathbb{R}^2 \to \mathbb{R}^4\)，对应颜色与光强；
  - 训练后生成稠密点云，再通过**凸包运算**转换为几何多面体 \(Fx ≤ b\)。

### 4. 安全路径规划算法（Algorithm 1）
- 输入：系统动力学( A,B )、NeRF 表征的障碍物与目标多面体集。  
- 主流程：
  1. 将 NeRF 体积分转换为约束多面体（障碍、目标、机器人）；
  2. 通过**Minkowski和**计算配置空间；
  3. 在滚动时域控制（MPC）框架下迭代计算可达集 \(R(τ)\)；
  4. 优化问题：
     \[
     \min d(G, R(τ)) \quad s.t.\ R(τ)∩O_i=∅,\ \dot{x}=Ax+Bu
     \]
  5. 若可达集与障碍分离，则该规划序列满足安全约束。
