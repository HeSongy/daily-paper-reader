## 摘要
现有的视觉语言导航（Vision-Language Navigation, VLN）方法通常采用以自我为中心的逐步导航范式，这种方式容易出现误差累积并限制效率。尽管近期方法尝试利用预先构建的环境地图，但它们常依赖于逐步更新的记忆图或对离散路径方案进行评分，这限制了连续的空间推理，并造成离散的瓶颈。我们提出了自上而下视觉语言导航（Top-Down VLN, TD-VLN），将导航重新表述为在预先构建的俯视地图上的一步全局路径规划问题，并配套我们新构建的 R2R-TopDown 数据集。为解决这一问题，我们引入 NavOne，这是一种统一框架，能够在单一端到端前向传递中直接预测多模态地图上的密集路径概率。NavOne 包含一个自上而下地图融合器，用于联合多模态地图表示，并扩展了注意力残差（Attention Residuals）以实现空间感知的深度混合。在 R2R-TopDown 上进行的大量实验表明，NavOne 在基于地图的 VLN 方法中实现了最先进的性能，其规划阶段的速度相比现有基于地图的基线方法提升了 8 倍，相比自我中心方法提升了 80 倍，从而实现了高效的全局导航。

## Abstract
Existing Vision-Language Navigation (VLN) methods typically adopt an egocentric, step-by-step paradigm, which struggles with error accumulation and limits efficiency. While recent approaches attempt to leverage pre-built environment maps, they often rely on incrementally updating memory graphs or scoring discrete path proposals, which restricts continuous spatial reasoning and creates discrete bottlenecks. We propose Top-Down VLN (TD-VLN), reformulating navigation as a one-step global path planning problem on pre-built top-down maps, supported by our newly constructed R2R-TopDown dataset. To solve this, we introduce NavOne, a unified framework that directly predicts dense path probabilities over multi-modal maps in a single end-to-end forward pass. NavOne features a Top-Down Map Fuser for joint multi-modal map representation, and extends Attention Residuals for spatial-aware depth mixing. Extensive experiments on R2R-TopDown show that NavOne achieves state-of-the-art performance among map-based VLN methods, with a planning-stage speedup of 8x over existing map-based baselines and 80x over egocentric methods, enabling highly efficient global navigation.

## 一、研究背景与核心问题

- **研究动机**：  
  视觉语言导航（Vision-Language Navigation, VLN）是具身智能（Embodied AI）领域的重要任务，要求智能体理解自然语言指令并在真实或模拟环境中完成导航。  
  现有方法主要采用 **以自我为中心（egocentric） 的逐步决策范式**，即根据视觉观察与历史状态反复预测下一个动作。这导致：
  - 误差在长程导航中不断累积；
  - 全局空间结构建模弱；
  - 长序列推理计算成本高（每步都需预测动作）。  
  尽管已有方法尝试引入预先构建的环境地图（例如使用记忆图或候选路径评分），但仍局限于 **离散式提案与局部搜索**，难以实现连续空间上的全局规划。

- **研究目标**：  
  作者提出一种新的任务设定，称为 **Top-Down VLN（TD-VLN）**，将导航转化为 **在预构建俯视地图上进行一步全局路径规划** 的问题，从而直接预测完整导航路径，而非逐步动作序列。

## 二、方法论：NavOne 框架

### 1. 核心思想
- **一次性全局规划（One-step global planning）**：  
  在多模态俯视图（RGB、占据图、语义图）和语言指令条件下，模型一次性预测整条路径及目标位置的概率分布，从而避免逐步决策误差。
  
- **联合多模态映射与空间注意机制**：  
  通过融合视觉（RGB）、几何（occupancy）、语义（semantic）以及语言特征，实现端到端路径推理。

### 2. 模型结构
**NavOne** 由三个主要组件构成：

1. **Top-Down Map Fuser（地图融合器）**：  
   - 输入包括三种俯视图：RGB 图、占据图（可通行区域）和语义图（41 类对象类别）。  
   - 通过通道拼接与语义嵌入，将它们映射为统一的特征张量。

2. **Path Former（编码–解码器结构）**：
   - 使用 ViT-B/16 视觉Transformer作为编码主干。  
   - 应用 **多层交叉注意力（cross-attention）** 将融合的视觉特征与语言特征对齐。  
   - 引入 **Attention Residuals（注意力残差）** 与 **Spatial-Aware Depth Query（空间感知深度查询）**：
     - Attention Residuals允许每个Transformer层访问所有前层输出，实现深层特征混合；
     - Spatial-Aware Depth Query使每个空间位置可以根据位置嵌入自适应地混合不同层级特征，提高空间分辨率。
   - 编码器输出经逐步上采样的解码器恢复至原始分辨率，输出两个密集图：
     - 路径概率图 \(P_{path} \in [0,1]^{H\times W}\)
     - 目标位置概率图 \(P_{goal} \in [0,1]^{H\times W}\)

3. **Path Extractor（路径提取器）**：  
   - 根据预测的概率图进行路径规划。  
   - 采用改进的 **A\*搜索算法**：
     - 将高路径概率区域作为低代价区域；
     - 将占据图中的障碍区域赋予高代价；
     - 从起点到目标点搜索最优路径；
     - 若目标点不可达，则调整至最近可达点重新规划。

### 3. 训练目标与优化
- 采用多任务损失函数：
  \[
  L_{total} = L_{path} + \alpha L_{goal} + \lambda L_{continuity}
  \]
  其中：
  - \(L_{path}, L_{goal}\)：二值交叉熵损失；
  - \(L_{continuity}\)：路径连续性约束，包括梯度平滑项 \(L_{grad}\)、起点约束 \(L_{start}\)、形态学连通性约束 \(L_{erosion}\)。  
  - 超参数：\(\alpha = 5.0, \lambda = 0.1\)。

## 三、实验设计与比较

### 1. 数据集与场景
- **R2R-TopDown 数据集**（本文新构建）：
  - 基于 Matterport3D 场景与 R2R-CE 任务转换而来；
  - 包含多模态俯视图（RGB、占据、语义）；
  - 过滤为单层场景，共三部分数据：  
    - 训练集 6,196 条；
    - 验证集（Seen）439 条；
    - 验证集（Unseen）1,003 条。
  - 每条样本包含语言指令（约26.8词）及对应轨迹。

### 2. 对比方法（Benchmark）
与多种基于地图的VLN方法比较：
- **WS-MGMap (2022)**：弱监督多粒度地图学习；
- **MapNav (2025)**：基于语义地图的记忆导航；
- **IPPD (2025)**：基于扩散模型的路径规划；
- 同时与代表性的 **逐步方法 ETPNav (2024)** 对比计算效率。

### 3. 评价指标
采用标准VLN指标：
- **SR（Success Rate）**
- **SPL（Success weighted by Path Length）**
- **TL（Trajectory Length）**
- **NE（Navigation Error）**

## 八、不足与局限

- **环境要求高**：  
  依赖 **预构建静态地图**，难以适应动态环境或在线SLAM变动。

  使用 Matterport3D 41 类语义标签，无法覆盖细粒度物体（如照片、标志），限制部分指令理解能力。

  虽做了噪声鲁棒性实验，但仍需验证在 **真实、动态、多层场景** 中的灵活性。

- **扩展性有限**：  
  当前实验仅在单层室内环境中进行；多层或室外场景尚未测试。
