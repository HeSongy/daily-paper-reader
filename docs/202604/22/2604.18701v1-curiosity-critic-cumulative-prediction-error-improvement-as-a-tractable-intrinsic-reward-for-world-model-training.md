## 摘要
基于局部预测误差的好奇心奖励仅关注当前的状态转移，而没有考虑世界模型在所有访问过的转移中的累积预测误差。我们提出了 Curiosity-Critic，它将内在奖励建立在该累积目标改进的基础上，并展示了这种方法可以化简为可处理的逐步形式：当前预测误差与当前状态转移的渐近误差基线之间的差值。我们通过与世界模型共同训练的学习型评论器在线估计这一基线；该评论器回归一个标量值，在世界模型饱和前即可收敛，从而在无需先验噪声底知识的情况下，将探索重新引导至可学习的转移。对于可学习的转移，该奖励较高；对于随机性强的转移，则趋近于基线，有效地在线区分了认知性（可约）和随机性（不可约）的预测误差。此前的基于预测误差的好奇心设计（从 Schmidhuber 1991 到学习特征空间的变体）可视为该基线特定近似的特例。在随机网格世界实验中，Curiosity-Critic 在收敛速度和最终世界模型精度上均优于预测误差和访问计数基线方法。

## Abstract
Local prediction-error-based curiosity rewards focus on the current transition without considering the world model's cumulative prediction error across all visited transitions. We introduce Curiosity-Critic, which grounds its intrinsic reward in the improvement of this cumulative objective, and show that it reduces to a tractable per-step form: the difference between the current prediction error and the asymptotic error baseline of the current state transition. We estimate this baseline online with a learned critic co-trained alongside the world model; regressing a single scalar, the critic converges well before the world model saturates, redirecting exploration toward learnable transitions without oracle knowledge of the noise floor. The reward is higher for learnable transitions and collapses toward the baseline for stochastic ones, effectively separating epistemic (reducible) from aleatoric (irreducible) prediction error online. Prior prediction-error curiosity formulations, from Schmidhuber (1991) to learned-feature-space variants, emerge as special cases corresponding to specific approximations of this baseline. Experiments on a stochastic grid world show that Curiosity-Critic outperforms prediction-error and visitation-count baselines in convergence speed and final world model accuracy.

## 一、研究动机与问题背景

- **背景**：强化学习中，模型驱动（model-based）方法依赖世界模型（world model）预测环境动态，以提升样本效率与泛化能力。但世界模型的训练质量直接限制其性能：若模型学习到不准确的状态预测，会导致规划误差累积。
- **问题**：
  - 传统的好奇心驱动方法（如 Schmidhuber 1991 的 *Curiosity V1/V2*）多基于**单步预测误差**，仅考虑当前状态转移。
  - 这类方法难以考虑**跨时间的累积预测误差改善**，也无法有效区分：
    - **可约误差（epistemic error）**：可通过学习减少；
    - **不可约误差（aleatoric error）**：由环境随机性决定。
  - 在存在高噪声的环境（如“noisy TV”问题）中，这些方法往往错误地被高噪声吸引，导致模型训练退化。
- **研究目标**：  
  作者旨在设计一个新的内在奖励机制，使智能体能判断某一转移是否具有“可学习性”，从而更高效地探索、提升世界模型学习质量。

## 二、方法论与算法机制

### 1. 核心思想
- 提出 **Curiosity-Critic**：
  - 将好奇心奖励定义为世界模型**累积预测误差的改进量**；
  - 推导出其可化简的逐步形式，使原本不可计算的全历史目标转化为一个**当前预测误差与渐近误差基线之差**：
    \[
    r_t = e(s_t,a_t|\theta_t) - \mathbb{E}_P[e(s_t,a_t|\theta_\infty)]
    \]
- 该形式意味着：  
  - 当状态转移仍可学习时（误差高但噪声低），奖励为正；  
  - 当转移已趋于不可学习的随机噪声区间，误差与基线相近，奖励接近零。

### 2. 关键技术与推导
- 从定义世界模型在历史数据集 \( D_t \) 上的累积误差改善出发，构造理论形式：
  \[
  r_t(D_t) = \sum_{(s',a') \in D_t} [e(s',a'|\theta_t) - e(s',a'|\theta_{t+1})]
  \]
- 通过**望远求和(telescoping sum)** 推导证明，该累积改进可完全等价为逐步奖励：
  \[
  r_t^{eff} = e(s_t,a_t|\theta_t) - e(s_t,a_t|\theta_{T+1})
  \]
- 再通过两个逼近：
  1. 用 \(\theta_\infty\) 代替训练终点的参数；
  2. 将单次样本误差替换为期望误差；
  得到上述最终形式。

### 3. 在线估计“渐近误差基线”
- 引入**Critic网络**（Curiosity-Critic）来近似 \( \mathbb{E}_P[e(s_t,a_t|\theta_\infty)] \)，估计当前转移的不可约噪声。
- Critic 与世界模型同步训练，训练流程如下（Algorithm 1）：
  1. 执行动作，观测下一个状态；
  2. 计算当前预测误差；
  3. 更新世界模型；
  4. 用更新后的误差作为 Critic 的回归目标；
  5. Critic 给出新的基线估计；
  6. 奖励 = 当前误差 − 基线估计；
  7. 用该奖励更新强化学习策略。
- 当 Critic 收敛速度快于世界模型时，它能提前识别“可学习区域”，增强探索效率。

## 三、实验设计

### 1. 环境与数据设置
- **环境**：30×30 的**随机网格世界（stochastic grid world）**。
  - 左半部分（15列）：完全**确定性区域（learnable）**；
  - 右半部分（15列）：**随机噪声区域（unlearnable）**（即“noisy TV”）。
  - 每格含一个 200 维二进制观测向量。
- **任务目标**：引导智能体尽快探索并学习确定性区域，从而最小化世界模型在可学习部分的预测误差。

### 2. 模型设置
- **世界模型（World Model θ）**：  
  两层 MLP，输入 60 维（行列 one-hot 拼接），输出 200 维；使用 MSE 损失。
- **Critic（ϕ）**：
  - 神经评估器版：输入 60 → 128 ReLU → 1（非负输出）；
  - 对比试验中还包括表格式 Critic（Tabular EMA）。
- **策略（Policy）**：基于 V 表的贪婪策略，加入 0.3 随机探索概率。

### 3. 对比方法（Baselines）
- Curiosity V1（预测误差奖励）；
- Curiosity V2（一步误差改善）；
- Visitation Count（基于访问计数的探索奖励）；
- Random Exploration（无内在奖励）；
- Oracle Critic（使用真实噪声基线的理想上界）。

### 4. 评估指标
- **主指标**：  
  学习过程中世界模型在确定性格子上的平均 L2 误差；
- **次指标**：  
  智能体在确定性区域的访问比例、收敛速度及收敛稳定性。
