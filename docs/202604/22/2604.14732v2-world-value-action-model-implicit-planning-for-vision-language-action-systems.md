## 摘要
视觉-语言-行动（Vision-Language-Action, VLA）模型已成为构建能够将感知与语言落地为行动的具身智能体的一种有前景的范式。然而，大多数现有方法依赖于直接动作预测，缺乏在长时间跨度的轨迹上进行推理并评估其后果的能力，从而限制了其在复杂决策任务中的表现。在本研究中，我们提出了 World-Value-Action（WAV）模型——一种在 VLA 系统中实现隐式规划的统一框架。与显式轨迹优化不同，WAV 模型学习一种结构化的潜在表征，用以描述在视觉观测和语言指令条件下的未来轨迹。学习到的世界模型预测未来状态，而轨迹价值函数评估其长时效用。动作生成被表述为在此潜在空间中的推理过程，模型逐步将概率质量集中在高价值且动态可行的轨迹上。我们从理论角度指出，直接在动作空间中进行规划会随着计划范围的增加而出现可行轨迹概率的指数衰减。相比之下，潜在空间中的推理能够重新塑造搜索分布，使其集中于可行区域，从而实现高效的长时决策。大量的模拟和真实世界实验表明，WAV 模型在任务成功率、泛化能力和鲁棒性方面均显著优于现有最先进的方法，尤其在长时和组合性场景中表现突出。相关代码可在 https://github.com/Win-commit/WAV 获取。

## Abstract
Vision-Language-Action (VLA) models have emerged as a promising paradigm for building embodied agents that ground perception and language into action. However, most existing approaches rely on direct action prediction, lacking the ability to reason over long-horizon trajectories and evaluate their consequences, which limits performance in complex decision-making tasks. In this work, we introduce World-Value-Action (WAV) model, a unified framework that enables implicit planning in VLA systems. Rather than performing explicit trajectory optimization, WAV model learn a structured latent representation of future trajectories conditioned on visual observations and language instructions. A learned world model predicts future states, while a trajectory value function evaluates their long-horizon utility. Action generation is then formulated as inference in this latent space, where the model progressively concentrates probability mass on high-value and dynamically feasible trajectories. We provide a theoretical perspective showing that planning directly in action space suffers from an exponential decay in the probability of feasible trajectories as the horizon increases. In contrast, latent-space inference reshapes the search distribution toward feasible regions, enabling efficient long-horizon decision making. Extensive simulations and real-world experiments demonstrate that the WAV model consistently outperforms state-of-the-art methods, achieving significant improvements in task success rate, generalization ability, and robustness, especially in long-horizon and compositional scenarios. Code is available at https://github.com/Win-commit/WAV.

## 一、研究背景与核心问题

- **研究动机**：  
  视觉-语言-行动（Vision-Language-Action, VLA）模型旨在让具身智能体理解视觉环境与语言指令并据此执行动作。然而，现有主流方法通常采用**直接动作预测**的方式——仅根据当前观测输出下一步动作。这种架构在面对**长时任务（long-horizon tasks）**时存在显著缺陷：  
  - 缺乏在未来轨迹上的推理能力；  
  - 无法评估多步行动的后果；  
  - 在复杂物理场景中难以保持连贯的动作序列。  
- **核心问题**：  
  如何在不显式依赖模型预测控制（MPC）或强化学习优化的前提下，让VLA模型具备**长时规划与隐式推理能力**。  

## 二、方法论：World–Value–Action (WAV) 模型

### 1. 核心思想
- 将规划视作一种推理过程，在**潜空间（latent space）**中而非显式动作空间中完成；
- 通过联合建模三个模块实现“预测—评估—行动”的闭环学习：  
  1. **World（世界模型）**：预测未来视觉状态；  
  2. **Value（价值模型）**：评估各个潜在轨迹的长时回报；  
  3. **Action（行动模块）**：在潜空间中逐步推理生成高价值、物理可行的动作序列。

### 2. 理论基础（主要结论）
- **Lemma 4.1**：在动作空间中进行搜索时，可行轨迹的概率随规划长度呈指数级衰减。  
- **Proposition 4.2**：潜空间采样通过重塑分布，将搜索集中到可行区域，其找到可行动轨迹的概率随时域长度呈指数增长。  
- **Corollary 4.3**：单次潜空间采样不足以获得最优解，需**迭代式推理**以不断聚焦高价值轨迹。

### 3. 模型结构
- **语言条件视频生成模块（World）**：  
  - 采用T5-XXL语言编码器结合DiT（Diffusion Transformer）生成多视角预测视频；  
  - 捕捉视觉历史、噪声变量和语言语义以形成未来视觉轨迹。  
- **轨迹价值模块（Value）**：  
  - 基于DiT的自回归结构，对生成的视觉轨迹估计累计折扣回报；  
  - 使用潜在价值标记（latent value token）进行跨模态交互。  
- **动作解码模块（Action）**：  
  - 将视频特征与价值特征结合，通过跨注意力生成可执行的机器人动作。  

### 4. 训练策略
- 采用三阶段的**Flow Matching**训练流程：
  1. 视频流训练（预测视觉未来）；  
  2. 价值流训练（预测回报信号）；  
  3. 动作流联合训练（生成最终动作）。  
- 奖励函数基于 ReinboT 框架，包含图像MSE/SSIM、关节与动作平滑性等多项密集奖励设计。

### 5. 潜空间迭代推理算法（Algorithm 1）
- 采用**迭代潜空间优化**：
  - 每轮采样视频潜变量与价值潜变量；
  - 通过信噪比（SNR）评估高价值轨迹；
  - 选择“精英样本（elite samples）”用于分布更新；
  - 利用平滑参数 α、β 防止分布坍塌；
  - 经过 K 轮迭代后，生成最优潜变量并解码为动作。
- 框架灵感来源于MPPI（Model Predictive Path Integral）控制。
