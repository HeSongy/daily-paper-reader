## 摘要
我们提出了三维锚定前瞻规划（3D-Anchored Lookahead Planning，简称3D-ALP），这是一种用于机器人操作的第二系统推理引擎，它结合了蒙特卡罗树搜索（Monte Carlo Tree Search，MCTS）与保持三维一致性的世界模型作为展开预测的算子。与仅根据当前摄像头画面评估动作的反应型策略不同，3D-ALP保留了一个持久的“摄像头到世界”（camera-to-world，c2w）锚点，能够在发生遮挡时仍然存续，从而实现对不再直接可见的物体位置的精确重新规划。在一个需要空间记忆的五步连续到达任务（实验 E3）中，3D-ALP在需要记忆的步骤上实现了0.650±0.109的成功率，而贪婪反应式基线仅为0.006±0.008（提升Δ=+0.645）；第5步成功率达到0.822，而贪婪策略则为0.000。消融研究（30个实验，3个随机种子）表明树搜索的空间记忆是主要驱动因素（提升+0.533，占总增益的82%），深层前瞻则提供了额外的收益（+0.111，占17%）。我们还识别并解决了将UCT-MCTS（应用于树的上置信界算法 [10]）用于连续机器人操作时的四种结构性失效模式。

## Abstract
We present 3D-Anchored Lookahead Planning (3D-ALP), a System 2 reasoning engine for robotic manipulation that combines Monte Carlo Tree Search (MCTS) with a 3D-consistent world model as the rollout oracle. Unlike reactive policies that evaluate actions from the current camera frame only, 3D-ALP maintains a persistent camera-to-world (c2w) anchor that survives occlusion, enabling accurate replanning to object positions that are no longer directly observable. On a 5-step sequential reach task requiring spatial memory (Experiment E3), 3D-ALP achieves 0.650 0.109 success rate on memory-required steps versus 0.006 0.008 for a greedy reactive baseline (Δ=+0.645), while step 5 success reaches 0.822 against 0.000 for greedy. An ablation study (30 episodes, 3 seeds) isolates tree search spatial memory as the primary driver (+0.533, 82% of gain) with additional benefit from deeper lookahead (+0.111, 17%). We also identify and resolve four structural failure modes in applying UCT-MCTS (Upper Confidence Bounds applied to Trees [10]) to continuous robotic manipulation.

## 一、研究背景与核心问题

### 研究动机
- 当前机器人操作系统多基于**视觉-语言-动作模型（VLA）**的直接映射：从当前摄像画面直接计算控制指令。
- 此类“System 1”型反应式策略在**单步任务**中表现良好，但在**多步连续任务和遮挡场景**中表现极差，因为缺乏：
  - **空间记忆（object permanence）**：无法记住物体在视野外的位置信息。
- 作者指出这些失败不是模型容量问题，而是**体系结构缺陷**——缺乏能跨步保持场景状态的持久机制。

### 研究整体含义
- 本文旨在为机器人操作引入一种“System 2”级的推理机制：能够在物体被遮挡、时间跨度较长的操作任务中维持空间记忆。
- 该方法**3D-Anchored Lookahead Planning（3D-ALP）**通过整合**世界模型**与**蒙特卡洛树搜索（MCTS）**，实现跨时间的路径规划与动作预测，显著提升多步任务的执行成功率。

## 二、方法论：核心思想与技术细节

### 核心思想
- 3D-ALP将机器人场景状态锚定在持续更新的**相机到世界坐标（Camera-to-World, c2w）**矩阵中，使得在目标物体被遮挡后仍可基于历史观测进行准确的重规划。
- 引入一个**三维一致性世界模型**（InSpatio-WorldFM），在MCTS展开过程中生成未来场景的“想象画面”。
- 采用混合评分机制，将**视觉语义与几何现实结合**，避免视觉模型对距离深度的错误判断。

### 技术结构（系统组件）
1. **Kinematic Bridge（前向运动学FK）**：根据机器人执行的每个动作更新c2w锚点；
2. **World Model Oracle（InSpatio-WorldFM）**：根据给定的c2w查询位置生成预测场景；
3. **Hybrid Geometric-Semantic Scorer**：融合视觉语义分数与实际三维距离惩罚；
4. **MCTS Engine**：基于上述预测与评分进行树搜索与回溯，从而挑选最优动作。

### 关键公式与机制（文字说明）
- **锚点更新机制**：  
  c2wₜ₊₁ = FK(qₜ₊₁)  
  每次执行动作后，通过前向运动学重新计算并更新摄像头在世界坐标下的位置。
- **世界模型融合防漂移机制**：  
  z_ref ← 0.7 × Enc(I_real) + 0.3 × z_ref  
  将真实画面编码与历史参考混合，以防止三维锚点漂移。
- **混合评分公式**：  
  S_total = S_semantic × max(0, 1 − ∥c2w − c2w_goal∥₂)  
  同时考虑视觉匹配与物理距离，使评分对几何不一致敏感。
- **MCTS四个结构修复点**：  
  - 排除零动作陷阱（防止停滞）；  
  - 深度衰减修正（保证多步前瞻）；  
  - 平均回溯替换为最大值回溯（防止好路径被稀释）；  
  - UCB常数重新标定（c=0.02，以适应连续评分）。
