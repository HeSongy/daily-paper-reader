## 摘要
临时无线网络呈现出复杂、固有且耦合的动态特性：节点移动、能量耗尽以及拓扑变化，这些特性使得解析建模十分困难。无模型的深度强化学习要求持续的在线交互，而现有的基于模型的方法采用扁平的状态表示，从而丧失了节点级结构。因此，我们提出了 G-RSSM，这是一种图结构的递归状态空间模型，能够维护节点级的潜在状态，并通过跨节点的多头注意力机制从离线轨迹中联合学习动态特性。我们将所提出的方法应用于下游任务——聚类，其中簇头选择策略完全通过在学习到的世界模型中的想象滚动进行训练。在涵盖 MANET、VANET、FANET、WSN 及战术网络的 27 个评估场景中（节点数 N=30 至 1000），即使仅在 N=50 上进行训练，所学习到的策略仍能保持较高的连通性。在本文中，我们提出了首个将多物理机制图结构世界模型应用于规模无关的无线自组网络中节点级组合决策的研究。

## Abstract
Ad hoc wireless networks exhibit complex, innate and coupled dynamics: node mobility, energy depletion and topology change that are difficult to model analytically. Model-free deep reinforcement learning requires sustained online interaction whereas existing model based approaches use flat state representations that lose per node structure. Therefore we propose G-RSSM, a graph structured recurrent state space model that maintains per node latent states with cross node multi head attention to learn the dynamics jointly from offline trajectories. We apply the proposed method to the downstream task clustering where a cluster head selection policy trains entirely through imagined rollouts in the learned world model. Across 27 evaluation scenarios spanning MANET, VANET, FANET, WSN and tactical networks with N=30 to 1000 nodes, the learned policy maintains high connectivity with only trained for N=50. Herein, we propose the first multi physics graph structured world model applied to combinatorial per node decision making in size agnostic wireless ad hoc networks.

## 二、方法论：图结构递归状态空间模型（G-RSSM）

### 1. 核心思想
- 在标准 RSSM（Recurrent State Space Model）的基础上，引入**图结构建模**与**节点级潜变量表示**。
- 以 **GNN（Graph Neural Network）+ GRU + 多头注意力（MHA）** 相结合的方式，捕获节点间交互与时间演化。
- 学习到的 G-RSSM 充当“数字孪生”（digital twin），可在离线阶段模拟网络的动态变化，从而实现**全“想象”式策略训练**。

### 2. 关键技术结构
- **图编码器（GATv2 Encoder）**：
  - 输入：每个节点7维特征（位置、速度、能量、度、簇头状态、生存状态、簇ID）。
  - 采用2层GATv2结构（4个注意力头，LayerNorm + ELU + dropout 0.1），输出每节点64维嵌入。
- **节点级递归状态更新**：
  - 每个节点持有独立 GRU（32维隐状态），输入包括上一步潜变量 \( z_{t-1}^i \) 与上一步动作 \( a_{t-1}^i \)。
  - 更新后通过多头自注意力实现跨节点信息交流：
    \[
    h_t^i = \text{LayerNorm}(\tilde{h}_t^i + \text{MHA}(\tilde{H}_t, \tilde{H}_t, \tilde{H}_t)_i)
    \]
- **潜变量建模**：
  - 使用DreamerV3风格的**分解式离散潜变量**（8个独立8分类因子）。
  - 采用先验与后验网络 \( p(z_t|h_t), q(z_t|h_t, o_t) \)，支持想象推理。
- **解码器头**：
  - 分别重建位置、能量、邻接矩阵、奖励与“continue”信号。
  - 训练损失：
    \[
    L_{WM} = L_{pos} + L_{energy} + 0.1L_{adj} + L_{reward} + L_{cont} + \beta \max(D_{KL}, \eta)
    \]
- **策略网络（Actor-Critic）**：
  - 基于节点特征 \([h_i, z_i, a_{t-1}^i]\) 学习分布式簇头（CH）选择。
  - 通过多轮迭代注意力（R=3 Refinement Rounds）实现协调优化。
  - 策略在虚拟世界模型中以PPO目标：
    \[
    J_{actor} = \mathbb{E}[\min(r_t(\theta)\hat{A}_t, \text{clip}(r_t(\theta),1\pm\epsilon)\hat{A}_t)] + \lambda_H H[\pi]
    \]
    并结合DreamerV3的连续性预测折扣。

## 五、主要结论与发现

- G-RSSM 可准确学习节点移动、能量衰减与拓扑演化的耦合关系，通过“想象滚动”进行策略训练。
- 所得 WM-Cluster 策略：
  - 在50节点场景中显著减少簇头切换（降75%–80%）；
  - 网络生命周期延长4.7倍；
  - 能量均衡性（Jain’s指数0.847）优于所有拓扑基方法；
  - 在未重新训练的情况下，对不同规模网络仍维持平均连通性0.969；
  - 验证了模型的**可扩展性（Size-agnostic）与泛化性**。
- G-RSSM 能作为物理环境模拟的**可微数字孪生替代品**，大幅降低仿真成本。
