## 摘要
越野自主导航需要可靠的三维感知能力，以便在复杂、非结构化的地形中实现稳健的障碍物检测。虽然激光雷达（LiDAR）具有高精度，但其成本高且功耗大。利用基础模型进行单目深度估计提供了一种轻量级的替代方案，但其在户外导航系统中的集成尚未得到充分研究。本文提出一个开源的越野导航系统，支持激光雷达与单目三维感知的联合应用，无需针对特定任务进行训练。在单目配置中，我们将零样本深度预测（Depth Anything V2）与基于稀疏 SLAM 测量（VINS-Mono）的度量深度重标定相结合。两项关键改进提升了系统的稳健性：边缘遮罩以减少障碍幻觉，以及时间平滑以减轻 SLAM 不稳定性的影响。生成的点云用于构建以机器人为中心的 2.5D 高程图，用于基于代价地图的路径规划。在逼真的仿真（Isaac Sim）和真实的非结构化环境中进行评估时，单目配置在大多数场景下与高分辨率激光雷达性能相当，显示出基于基础模型的单目深度估计在稳健越野导航中可作为激光雷达的可行替代方案。通过开源该导航系统及仿真环境，我们提供了一个完整的越野导航管线以及可复现的基准。代码可在 https://github.com/LARIAD/Offroad-Nav 获取。

## Abstract
Off-road autonomous navigation demands reliable 3D perception for robust obstacle detection in challenging unstructured terrain. While LiDAR is accurate, it is costly and power-intensive. Monocular depth estimation using foundation models offers a lightweight alternative, but its integration into outdoor navigation stacks remains underexplored. We present an open-source off-road navigation stack supporting both LiDAR and monocular 3D perception without task-specific training. For the monocular setup, we combine zero-shot depth prediction (Depth Anything V2) with metric depth rescaling using sparse SLAM measurements (VINS-Mono). Two key enhancements improve robustness: edge-masking to reduce obstacle hallucination and temporal smoothing to mitigate the impact of SLAM instability. The resulting point cloud is used to generate a robot-centric 2.5D elevation map for costmap-based planning. Evaluated in photorealistic simulations (Isaac Sim) and real-world unstructured environments, the monocular configuration matches high-resolution LiDAR performance in most scenarios, demonstrating that foundation-model-based monocular depth estimation is a viable LiDAR alternative for robust off-road navigation. By open-sourcing the navigation stack and the simulation environment, we provide a complete pipeline for off-road navigation as well as a reproducible benchmark. Code available at https://github.com/LARIAD/Offroad-Nav.

## 一、研究问题与背景

- **核心问题**：在越野场景中实现可靠的自主导航，需要高质量的三维感知以应对非结构化地形与复杂障碍物。传统方案多依赖 **激光雷达（LiDAR）**，但其高成本、高功耗以及易被探测的特性限制了应用的普及。

- **研究动机**：  
  鉴于激光雷达的物理与经济约束，研究者希望探索 **单目深度估计（Monocular Depth Estimation）** 的可能性。近年来基础模型（Foundation Models）在零样本深度估计方面取得突破，为无需特定训练的视觉感知提供了新机会，但其在室外、动态越野导航中的集成尚欠系统研究。

- **总体目标**：构建一个 **同时支持 LiDAR 和单目视觉感知的开源越野自动导航栈（Navigation Stack）**，验证单目深度估计在复杂场景中代替 LiDAR 的可行性。

## 二、方法论：系统架构与关键技术

### 1. 整体架构

论文提出一个完整的三阶段自主导航管线：
1. **3D 感知模块（3D Perception）**
2. **地面分割与障碍检测（Ground Segmentation & Obstacle Detection）**
3. **路径规划模块（Path Planning）**

系统输入可以是：
- 激光雷达点云；
- 或单目相机图像 + SLAM（VINS-Mono）稀疏点云。

输出为机器人运动控制命令（速度指令）。

### 2. 单目深度估计与尺度校正

- 使用 **Depth Anything V2** 模型进行零样本深度预测，生成归一化的深度视差图。  
- 为转化为**度量深度（metric depth）**，采用与 **VINS-Mono** SLAM 输出的稀疏深度点进行**尺度和偏移量回归**：
  \[
  \tilde{s}_t = \alpha \tilde{s}_{t-1} + (1-\alpha)\hat{s}_t
  \]
  其中，α 为指数平滑系数（设为 0.8），用于缓解 SLAM 不稳定带来的尺度抖动。

### 3. 鲁棒性优化

- **边缘遮罩（Edge Masking）**：对预测视差图应用 Sobel 边缘滤波，屏蔽靠近深度不连续处的像素（5 像素范围），以减轻“幻觉障碍物”影响。  
- **时间平滑（Temporal Smoothing）**：对尺度参数进行指数移动平均，提高跨帧一致性。

### 4. 地面分割与映射

- 点云经 **Cloth Simulation Filter (CSF)** 处理，将点云反转并模拟“布料落地”分离地面与障碍物，生成 **2.5D 高程图**。  
- 基于高度阈值（默认 30cm）生成 **二值代价地图（Costmap）**。  
- 路径规划采用：  
  - **全局规划器**：A* 算法；  
  - **局部规划器**：Timed-Elastic-Bands (TEB)，优化连续路径并考虑障碍膨胀（inflation）。

## 六、主要结论与发现

- **单目深度估计（Mono-VINS）性能与激光雷达相当**：  
  - 在多数仿真任务中，成功率与 LiDAR 持平甚至更优（最高 97% SR）。  
  - 实际场景中与 LiDAR 成功率均为 100%，距离比 1.0，仅路径效率略低 20%。

- **关键改进有效性**：边缘遮罩与时间平滑带来 SPL 平均提升约 21%。

- **复杂地形问题**：单目方案在“高草场景”失败率较高，易误判为障碍。

## 七、优点与创新

- **完全开源的越野导航栈**（含仿真和算法模块）。  
- **无需额外训练**：依赖视觉基础模型 + 几何校正即可部署。  
- **双感知模式兼容**：可在不同硬件之间切换。  
- **轻量化与低功耗**：单目系统功耗约为 LiDAR 的 1/10（2W vs 20W）。  
- **丰富、系统的实验设计**：跨仿真与真实平台验证，具备复现性与开放性。  
- **实时性优化**：使用 TensorRT 和 FP16 加速，实现嵌入式推理。

## 八、不足与局限

- **视觉误判问题**：高草等柔性障碍在深度估计中边缘模糊，导致导航失败。  
- **依赖 SLAM 稳定性**：若 SLAM漂移或失效，尺度校正不准。  
- **计算负载较高**：单目推理频率仅为 LiDAR 一半，实时性受限。  
- **参数敏感性**：CSF 与 TEB 需针对不同地形手动调参。  
- **尚未支持 ROS2**：目前实现基于 ROS1，后续计划迁移。  
- **未涵盖负障碍物（坑洞）或可通行分类模块**，未来需扩展 Traversability 分析。
