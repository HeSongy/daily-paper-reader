## 摘要
具身智能体的世界模型在给定智能体采取的动作条件下预测未来的观测结果。对于复杂的具身体，动作空间维度很高且难以明确指定：例如，要精确控制一个类人体智能体，需要为其每个关节的运动进行具体指定。这使得世界模型难以控制，并且基于搜索的方法（如CEM）在高维动作空间中扩展性较差，因而规划计算代价高昂。为了解决这一问题，我们训练了一个轻量级策略，将高层动作映射为低层关节动作序列。将该策略与冻结的世界模型组合后，得到一个提升后的（lifted）世界模型，该模型可从单个高层动作预测出一系列未来观测。我们在类人体具身体上实现了此框架，将高层动作空间定义为当前观测帧上标注的一小组二维路径点，每个路径点为一个末端关节（骨盆、头部、双手）指定一个短期目标位置。路径点维度低、视觉上可解释且易于人工指定或用于搜索。结果表明，该提升后的世界模型在保持计算效率的同时，显著优于直接在低层关节空间中搜索的基线方法（达到目标姿态的平均关节误差降低3.8倍），并能推广到策略未见过的环境中。

## Abstract
World models of embodied agents predict future observations conditioned on an action taken by the agent. For complex embodiments, action spaces are high-dimensional and difficult to specify: for example, precisely controlling a human agent requires specifying the motion of each joint. This makes the world model hard to control and expensive to plan with as search-based methods like CEM scale poorly with action dimensionality. To address this issue, we train a lightweight policy that maps high-level actions to sequences of low-level joint actions. Composing this policy with the frozen world model produces a lifted world model that predicts a sequence of future observations from a single high-level action. We instantiate this framework for a human-like embodiment, defining the high-level action space as a small set of 2D waypoints annotated on the current observation frame, each specifying a near-term goal position for a leaf joint (pelvis, head, hands). Waypoints are low-dimensional, visually interpretable, and easy to specify manually or to search over. We show that the lifted world model substantially outperforms searching directly in low-level joint space ($3.8\times$ lower mean joint error to the goal pose), while remaining more compute-efficient and generalizing to environments unseen by the policy.

## 一、核心问题与研究背景

- **研究动机**：  
  具身智能体（例如类人机器人）需要世界模型（world model）来预测在采取某一动作后的未来观测，用于理解和控制物理世界。  
  随着智能体具身体复杂度提升，**动作空间维度急剧增高**（数十个关节参数），传统搜索式规划方法（如 **CEM, Cross-Entropy Method**）在高维空间中计算量爆炸，导致规划效率低且难以控制。  

- **核心问题**：  
  如何在保持世界模型预测能力的同时，**降低动作空间维度，提高规划效率与控制可解释性**。

- **研究意义**：  
  提出一种“**提升（Lifting）世界模型**”的层次化设计，将高维关节动作抽象为低维、高层的视觉控制目标，使得世界模型能够更高效地用于规划与决策。

## 二、方法论：核心思想与技术细节

### 核心思想
- 研究提出一种 **“Lifted World Model (LWM)”**。
- 通过训练一个轻量级的 **高层动作到低层动作映射策略**（goal-conditioned policy），将高层动作（如图像中的二维“路径点”waypoints）转换成一段低层关节动作序列。
- 冻结已有的 **低层世界模型（如 PEVA）**，与该策略组合实现新的可规划模型。

### 技术流程
1. **高层动作定义（High-Level Action Space）**：  
   - 在当前观测图像上标注少量 **二维路径点**（pelvis、head、left-hand、right-hand），表示短期动作目标。  
   - 训练时由未来真实姿态经 3D→2D 投影获得；推理时可由人类指定或在 CEM 搜索中生成。

2. **策略学习（Policy Training）**：  
   - 训练一个 **扩散式策略（Diffusion Policy）**，实现从带路径点标注的观察输入到低层关节动作序列的生成。  
   - 输入：图像序列、姿态序列、waypoints；输出：未来动作序列。  
   - 使用 DINOv3-S 视觉编码器 + Transformer 结构，添加 3D 位置嵌入与 UNet 去噪模块。

3. **Lifted World Model (LWM) 构建**：  
   - 将预测策略与冻结的世界模型组合，在一个高层动作输入下，生成多步未来观测。  
   - 公式化：  
     - 动作生成： \( a^{LL}_{t:t+T-1} = \pi_\theta(o_t, p_t, a^{HL}_t) \)  
     - 世界模型迭代： \( o_{t+1} = f_\phi(o_t, a^{LL}_t) \)  
     - 综合得： \( o_{t+T} = f^{HL}(o_t, p_t, a^{HL}_t) \)

4. **规划算法（Planning via LWM）**：  
   - 在高层空间中使用 CEM 搜索最优路径点序列。  
   - 优化目标为最小化生成观测与目标观测之间的感知距离或关节姿态误差。  
   - 由于高层空间维度仅为 8（较低层关节空间的 48），搜索效率大幅提升。

## 三、实验设计

- **数据集**：  
  - 使用 **Nymeria dataset**：包含来自 Project Aria 眼镜与 XSens 动作捕捉系统的第一人称视频与人体姿态数据。  
  - 约 50 种室内和室外场景，支持混合导航和交互任务。

- **低层世界模型基线**：  
  - 使用 **PEVA** (Whole-body conditioned Egocentric Video Prediction) 作为冻结的低层模型。

- **对比方法基线**：  
  1. Unconditioned Policy（无目标条件）  
  2. Image-Conditioned Policy（使用目标图像）  
  3. PEVA CEM（直接在低层关节空间搜索）  
  4. Lifted CEM (2D Waypoints)  
  5. Lifted CEM (3D Waypoints)

- **评估指标**：  
  - **Mean Joint Error (MJE)**：预测末端关节与目标姿态的空间距离（米）。  
  - 分别计算叶关节、中间关节、整体平均误差。

- **消融与扩展实验**：  
  - 架构改进、姿态上下文、waypoint 条件、masking、3D depth条件等影响。  
  - 任务泛化测试：对训练时未见场景进行评估。
