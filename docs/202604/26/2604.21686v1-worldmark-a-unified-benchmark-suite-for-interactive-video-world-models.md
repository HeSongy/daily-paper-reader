## 摘要
交互式视频生成模型（如 Genie、YUME、HY-World 和 Matrix-Game）正在迅速发展，但每个模型都在其各自的基准上评估，使用的场景和轨迹各不相同，导致无法进行公平的模型间比较。现有的公共基准提供了有用的指标，比如轨迹误差、美学得分以及基于 VLM 的评判，但没有提供标准化的测试条件——相同的场景、相同的动作序列以及统一的控制接口——使得这些指标在输入异构的模型之间可比。我们提出了 WorldMark，这是首个为交互式图像到视频世界模型提供这种共同测试平台的基准。WorldMark 的贡献包括：(1) 一个统一的动作映射层，将共享的 WASD 风格动作词汇转换为各模型的原生控制格式，从而实现六个主要模型在相同场景和轨迹上的公平比较；(2) 一个分层的测试套件，包含 500 个评估案例，涵盖第一人称和第三人称视角、写实与风格化场景，以及三个难度层级（从简单到困难），持续时间从 20 到 60 秒；(3) 一个模块化的评估工具包，用于视觉质量、控制对齐和世界一致性，设计为研究者可以在保留标准化输入的同时，随着领域发展插入自己的指标。我们将发布所有数据、评估代码和模型输出以促进未来研究。除了离线评估指标之外，我们还推出了 World Model Arena（warena.ai），这是一个在线平台，任何人都可以在上面对领先的世界模型进行对战，并观看实时排行榜。

## Abstract
Interactive video generation models such as Genie, YUME, HY-World, and Matrix-Game are advancing rapidly, yet every model is evaluated on its own benchmark with private scenes and trajectories, making fair cross-model comparison impossible. Existing public benchmarks offer useful metrics such as trajectory error, aesthetic scores, and VLM-based judgments, but none supplies the standardized test conditions -- identical scenes, identical action sequences, and a unified control interface -- needed to make those metrics comparable across models with heterogeneous inputs. We introduce WorldMark, the first benchmark that provides such a common playing field for interactive Image-to-Video world models. WorldMark contributes: (1) a unified action-mapping layer that translates a shared WASD-style action vocabulary into each model's native control format, enabling apples-to-apples comparison across six major models on identical scenes and trajectories; (2) a hierarchical test suite of 500 evaluation cases covering first- and third-person viewpoints, photorealistic and stylized scenes, and three difficulty tiers from Easy to Hard spanning 20-60s; and (3) a modular evaluation toolkit for Visual Quality, Control Alignment, and World Consistency, designed so that researchers can reuse our standardized inputs while plugging in their own metrics as the field evolves. We will release all data, evaluation code, and model outputs to facilitate future research. Beyond offline metrics, we launch World Model Arena (warena.ai), an online platform where anyone can pit leading world models against each other in side-by-side battles and watch the live leaderboard.

- **问题**：不同模型在**私有数据集与场景**上进行评测，缺乏统一的**动作输入、场景设置与一致的控制接口**，导致跨模型之间的结果不可比较。

- **目标**：提出一个**标准化基准（benchmark）**，在相同视觉场景与统一动作控制下，对交互式从图像到视频（I2V）的“世界模型”进行可比评估。

## 二、方法论：WorldMark 框架设计

### 1. 核心思想
通过构建一个统一的评测平台，实现跨模型的公平比较。该平台包含：
1. **统一动作映射层（Unified Action Interface）**：  
   将标准化的“WASD + L/R”动作词汇映射到不同模型各自的控制接口（如文字描述、姿态参数、游戏手柄输入等），确保语义一致。
2. **分层测试套件（Hierarchical Test Suite）**：  
   提供多样化场景（写实与风格化、第一/第三人称视角）与不同难度动作轨迹（20秒–60秒），总计500个测试案例。
3. **模块化评估工具包（Modular Evaluation Toolkit）**：  
   从三个维度对结果视频进行评测：
   - **视觉质量（Visual Quality）**：基于 LAION 美学评分与 MUSIQ 图像质量。
   - **控制对齐（Control Alignment）**：基于 SLAM 重建的平移与旋转误差。
   - **世界一致性（World Consistency）**：通过重投影误差与基于 VLM（视觉语言模型）的时空一致性分析。

### 2. 关键技术细节

- **动作控制映射表**  
  每种模型定义不同输入，例如：
  | 模型 | 输入形式 | 映射方式 |
  |------|-----------|----------|
  | YUME | 文本提示 | 文本嵌入方向词 |
  | HY-World / HY-GameCraft | 6-DoF 姿态参数 | 姿态嵌入匹配 |
  | Genie 3 | 游戏手柄控制 | 按键信号 |
  | Matrix-Game | Action API | 直接函数调用 |
  | Open-Oasis | 25维连续动作向量 | 向量维度对应控制方向 |

- **评估公式（文字形式）**
  - **平移误差**：基于SLAM重建的相机中心距离（尺度不变的欧氏距离）。
  - **旋转误差**：预测与真实旋转矩阵间的几何角度差。
  - **重投影误差**：场景3D点经SLAM优化后的投影偏差。
  - **一致性检测**：基于VLM对帧序列进行语义一致性分析，判断对象形态、风格与内容是否稳定。

- **场景与动作生成**
  - 场景图像采集自 WorldScore 数据集，经 VLM 判定物理合理性。
  - 每个场景自动挑选5个合理动作序列（共15种标准动作模板）。

- **评估工作流**
  1. 选定场景与视觉风格；
  2. 统一动作翻译为模型输入；
  3. 模型生成视频；
  4. 自动计算三类指标。

## 六、主要结论与发现

1. **视觉质量与世界一致性几乎不相关**：  
   - YUME 1.5 生成最美观的帧（高美学分），但长期一致性较差；
   - Genie 3 视觉略逊，却保持最佳长期空间与风格一致性。
2. **精准控制不等于整体质量**：  
   - HY-Game 动作对齐最准确，但画面质量较低；
   - Genie 3 控制误差略高却具有更稳定的世界表现。
3. **第三人称是普遍薄弱点**：  
   - Matrix-Game 在第三人称下旋转误差放大约 20 倍，显示当前模型难以在有角色视角中保持控制稳定。
4. **领域迁移困难**：  
   - Open-Oasis 在Minecraft域表现良好，但在现实场景中性能崩溃。
5. **基准有效性验证**：  
   - 自动指标与人评高度一致，验证了该Benchmark的客观性与可靠性。
