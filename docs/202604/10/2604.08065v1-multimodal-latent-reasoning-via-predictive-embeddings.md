## 摘要
借助工具增强的多模态推理使视觉语言模型（VLM）能够通过与外部工具（例如裁剪、深度估计）的交互改进感知能力。然而，这类方法会带来较大的推理开销、需要专门的监督，并且容易出现错误的工具调用。我们提出了 Pearl（Predictive Embedding Alignment for Reasoning in Latent space，即潜空间推理的预测嵌入对齐），这是一个受 JEPA 启发的框架，能够完全在潜空间中从专家使用工具的轨迹中学习，从而在推理阶段无需显式调用工具。与基于重建的潜空间推理方法不同，后者通过自回归方式生成潜在 token，因而存在训练-推理不匹配以及对多步工具使用支持有限等问题，Pearl 则是直接从多模态轨迹中学习预测嵌入，并保持标准的视觉-语言生成流程：它与具体模型无关、训练简单，并且天然支持包含多个工具调用的轨迹。在多个感知基准测试中的实验表明，Pearl 能够达到或优于标准的监督微调和基于重建的潜空间推理方法的性能。此外，我们提供了实证证据表明，基于重建的方法主要学习的是嵌入，而非潜空间中的图像编辑，从而凸显了预测嵌入学习作为更为合理替代方案的潜力。

## Abstract
Tool-augmented multimodal reasoning enables visual language models (VLMs) to improve perception by interacting with external tools (e.g., cropping, depth estimation). However, such approaches incur substantial inference overhead, require specialized supervision, and are prone to erroneous tool calls. We propose Pearl (Predictive Embedding Alignment for Reasoning in Latent space), a JEPA-inspired framework that learns from expert tool-use trajectories entirely in the latent space, eliminating the need for explicit tool invocation at inference time. Unlike reconstruction-based latent reasoning methods, which autoregressively generate latent tokens and suffer from training-inference mismatch and limited support for multi-step tool use, Pearl directly learns predictive embeddings from multimodal trajectories while preserving the standard vision-language generation pipeline: it is model-agnostic, simple to train, and naturally supports trajectories with multiple tool calls. Experiments across multiple perception benchmarks show that Pearl matches or outperforms standard supervised fine-tuning and reconstruction-based latent reasoning approaches. Furthermore, we provide empirical evidence that reconstruction-based methods primarily learn embeddings rather than image edits in latent space, motivating predictive embedding learning as a more principled alternative.

## 一、核心问题与研究动机

- **研究背景：**  
  近年来，多模态推理（特别是视觉语言模型 VLM）通常通过调用外部视觉工具（如裁剪、目标检测、深度估计）执行感知增强。  
  这些“工具增强”方法虽能提升感知和推理精度，但存在显著问题：
  - 推理阶段的**计算开销大、延迟高**；
  - 需要**复杂的监督训练数据**以学习选择与配置工具；
  - 工具调用错误会污染上下文、浪费算力；
  - 对多类型、多步骤工具序列缺乏有效支持。

- **核心问题：**  
  如何让多模态模型在**不显式调用外部工具**的情况下，依然能够学习并复现这些工具带来的推理增益？

- **研究目标：**  
  本文提出 **PEARL（Predictive Embedding Alignment for Reasoning in Latent Space）**，在**潜空间（latent space）**中学习预测性嵌入，从专家工具使用轨迹中隐式学习工具的效果，使模型在推理时只需处理图像和文本，不再真正调用工具。

## 二、方法论：PEARL 框架

### 1. 核心思想
- 借鉴 **JEPA (Joint Embedding Predictive Architecture)** 的思想；
- 将图像-问题对 ⟨I₀, Q⟩ 与专家轨迹 R（含工具使用的图像与推理文本序列）视为同一任务的两个“视图”；
- 通过预测输入视图对应的**轨迹潜嵌入**，模型可在潜空间中学习到等价的工具使用影响；
- 推理阶段无需生成任何中间视觉结果或调用工具，仅依赖标准的语言模型解码。

### 2. 模型组成
- **输入编码 Enc(x)：**  
  将图像-问题对 ⟨I₀, Q⟩ 编码为隐藏状态。
- **轨迹编码 Enc(R)：**  
  将专家提供的多步图像与文本序列 (I₁, T₁, …, Iₙ, Tₙ) 编码为完整的轨迹表征。
- **预测器 Pred(·)：**  
  在输入编码基础上追加若干 [PRED] token，用原模型的自注意机制计算预测潜嵌入  
  \[
  \hat{h}_R = \text{Pred}(h_x)
  \]
- **JEPA 对齐损失（L\_JEPA）：**  
  以 SmoothL1 距离度量预测潜嵌入与真实轨迹嵌入的差异：
  \[
  L_{\mathrm{JEPA}} = D(\hat{h}_R, \mathrm{sg}[h_R])
  \]
  （其中 sg 表示停止梯度流）
- **Next-Latent 预测正则项（L\_NextLat）：**  
  使隐藏状态 ht 能预测未来潜状态 h_{t+i}，促进其具备“belief state”特性，提高序列表征稳定性。
- **标准语言生成损失（L\_VLM）：**  
  维持模型正常的自回归文本生成能力。
- **总训练目标：**
  \[
  L_{\mathrm{PEARL}} = L_{\mathrm{VLM}} + \lambda[L_{\mathrm{JEPA}} + L_{\mathrm{NextLat}}]
  \]

### 3. 推理流程
- 推理时仅输入图像与文字问题；
- 无需任何 [PRED] token、工具调用或潜空间解码；
- 模型直接生成答案，与普通 VLM 推理过程完全一致。

  - 因此，**预测性嵌入学习**是一种更原则、更高效的替代方法。
