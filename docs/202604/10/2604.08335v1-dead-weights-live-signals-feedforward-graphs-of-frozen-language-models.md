## 摘要
我们提出了一种前馈图结构架构，其中异质的冻结大型语言模型作为计算节点，通过学习得到的线性投影在共享的连续潜在空间中进行通信。基于最近的研究表明独立训练的 LLM 潜在空间在几何上具有兼容性（Armstrong 2026），我们将这一发现从静态的双模型引导扩展到端到端可训练的多节点图结构，其中投影矩阵通过残差流注入钩子的反向传播联合优化。三个小型冻结模型（Llama-3.2-1B、Qwen2.5-1.5B、Gemma-2-2B）将输入编码到共享的潜在空间中，其聚合信号被注入两个较大的冻结模型（Phi-3-mini、Mistral-7B），这些模型的表示进一步馈送到一个轻量级的交叉注意力输出节点。在仅有约 1760 万个可训练参数、约 120 亿个冻结参数的条件下，该架构在 ARC-Challenge 上取得 87.3%、在 OpenBookQA 上取得 82.8%、在 MMLU 上取得 67.2% 的成绩，分别比最佳的单个组成模型高出 11.4、6.2 和 1.2 个百分点，并比参数匹配的冻结单模型学习分类器高出 9.1、5.2 和 6.7 个百分点。通过多个冻结模型边界的梯度流被经验性验证为可行，且输出节点在无显式监督的情况下发展出跨二层节点的选择性路由行为。

## Abstract
We present a feedforward graph architecture in which heterogeneous frozen large language models serve as computational nodes, communicating through a shared continuous latent space via learned linear projections. Building on recent work demonstrating geometric compatibility between independently trained LLM latent spaces~\cite{armstrong2026thinking}, we extend this finding from static two-model steering to end-to-end trainable multi-node graphs, where projection matrices are optimized jointly via backpropagation through residual stream injection hooks. Three small frozen models (Llama-3.2-1B, Qwen2.5-1.5B, Gemma-2-2B) encode the input into a shared latent space whose aggregate signal is injected into two larger frozen models (Phi-3-mini, Mistral-7B), whose representations feed a lightweight cross-attention output node. With only 17.6M trainable parameters against approximately 12B frozen, the architecture achieves 87.3\% on ARC-Challenge, 82.8\% on OpenBookQA, and 67.2\% on MMLU, outperforming the best single constituent model by 11.4, 6.2, and 1.2 percentage points respectively, and outperforming parameter-matched learned classifiers on frozen single models by 9.1, 5.2, and 6.7 points. Gradient flow through multiple frozen model boundaries is empirically verified to be tractable, and the output node develops selective routing behavior across layer-2 nodes without explicit supervision.

## 二、方法论：架构与关键技术

### 1. 核心思想
- 构建一个**由若干冻结语言模型节点组成的前馈图（Feedforward Graph）**；
- 各节点之间的通信通过**共享连续潜在空间（shared latent space）**实现；
- 模型间信息传递依赖于**学习得到的线性投影矩阵**；
- 训练仅更新这些投影矩阵和最终的输出节点，其余模型权重全部冻结。

### 2. 架构细节
论文中的图结构分为三个主要层次：

#### （1）**第一层：多样化编码层**
- 使用三个小型冻结模型（Llama-3.2-1B、Qwen2.5-1.5B、Gemma-2-2B）。
- 每个模型接收相同输入，但带有不同分析提示（factual / reasoning / language framing）。
- 从其最后层提取隐藏状态，经 $L_2$ 归一化后通过各自的线性投影矩阵 $W_1, W_2, W_3$ 映射入一个维度为 1024 的共享潜空间：
  \[
  z_1 = \frac{1}{3}\sum_i W_i h_i
  \]

#### （2）**第二层：注入与精炼**
- 将共享潜向量 $z_1$ 注入到两个较大模型（Phi-3-mini、Mistral-7B）的残差流（residual stream）中，注入点位于模型层深 0.75 处（$l_S = 0.75$）。
- 注入比例 $\alpha = 0.25$，以控制混合强度：
  \[
  h' = (1 - \alpha)h_{node} + \alpha(\tilde{z_1}\|h_{node}\|_2)
  \]
- 注入后再提取每个模型的最终隐藏状态，经投影矩阵 $W_4, W_5$ 变换回共享空间。

#### （3）**输出节点：跨注意力聚合**
- 使用二维共享表示 $[z_2^{(4)}, z_2^{(5)}]$ 作为键和值，经过4头 Multi-Head-Attention (MHA)机制与学习的查询向量计算输出：
  \[
  \hat{y} = \text{softmax}(W_{cls}\,\text{MHA}(q, [z_2^{(4)}, z_2^{(5)}]))
  \]
- 该机制自发实现了**动态的模型间路由**。

### 3. 参数规模与训练机制
- 可训练参数：仅 **5个投影矩阵 + 1个输出节点**，共1760万参数；
- 冻结参数约 120 亿；
- 使用 **AdamW** 优化器，学习率分别为 $10^{-3}$（投影矩阵）与 $10^{-4}$（输出节点），余弦退火调度；
- 梯度通过冻结模型的注入点反向传播，完全端到端训练。
