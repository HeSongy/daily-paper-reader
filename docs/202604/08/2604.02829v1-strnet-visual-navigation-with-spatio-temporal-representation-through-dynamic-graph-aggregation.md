## 摘要
视觉导航要求机器人基于一系列第一人称视觉观测到达指定目标（如一幅图像）。尽管近期基于学习的方法取得了显著进展，它们通常侧重于改进策略头或决策策略，而依赖于简化的特征编码器和时间池化来表示视觉输入。这导致了细粒度空间和时间结构的丢失，从而限制了动作预测和进度估计的准确性。本文提出了一种统一的时空表示框架，用于增强视觉编码以实现机器人导航。我们的方法从图像序列和目标观测中提取特征，并通过设计的时空融合模块将它们融合。该模块在每一帧内执行空间图推理，并利用结合多分辨率差异感知卷积的混合时间移位模块来建模时间动态。实验结果表明，我们的方法能够稳定提升导航性能，并为目标条件控制提供一个具有可泛化性的视觉骨干网络。代码可在 \href{https://github.com/hren20/STRNet}{https://github.com/hren20/STRNet} 获取。

## Abstract
Visual navigation requires the robot to reach a specified goal such as an image, based on a sequence of first-person visual observations. While recent learning-based approaches have made significant progress, they often focus on improving policy heads or decision strategies while relying on simplistic feature encoders and temporal pooling to represent visual input. This leads to the loss of fine-grained spatial and temporal structure, ultimately limiting accurate action prediction and progress estimation. In this paper, we propose a unified spatio-temporal representation framework that enhances visual encoding for robotic navigation. Our approach extracts features from both image sequences and goal observations, and fuses them using the designed spatio-temporal fusion module. This module performs spatial graph reasoning within each frame and models temporal dynamics using a hybrid temporal shift module combined with multi-resolution difference-aware convolution. Experimental results demonstrate that our approach consistently improves navigation performance and offers a generalizable visual backbone for goal-conditioned control. Code is available at \href{https://github.com/hren20/STRNet}{https://github.com/hren20/STRNet}.

### 1.1 问题背景
视觉导航（visual navigation）要求机器人仅凭视觉输入完成目标到达任务，即根据连续的第一人称图像序列找到通向目标图像的路径。  
近年来的学习型导航方法（如 ViNT、NoMaD、NaviBridger）多将重点放在决策策略（policy head）或强化学习架构上，而忽视了视觉特征的高质量时空表征。

### 1.2 动机与问题
- 现有方法往往采用**简单的平均池化或时序RNN编码器**，导致空间几何和时间动态丢失；

- 时序建模中缺乏对局部运动线索的保持；
- 结果表现为“模糊”或“过平滑”的特征表示，削弱了动作精度与路径连贯性。  

因此，论文旨在改进视觉编码阶段，提出一种**统一的时空特征融合框架**，以增强导航过程中的空间理解与时间预测能力。

## 二、方法论与核心设计

### 2.1 方法总体思路
STRNet（Spatio-Temporal Representation Network）通过对**图像序列与目标图像特征的联合建模**，实现对空间和时间信息的显式融合。  
核心创新包括：
- **图结构化的空间特征聚合（Spatial Graph Aggregation）**
- **混合时移结合多分辨率差异卷积的时间融合（Hybrid Temporal Fusion）**
- **双任务头设计：动作扩散预测 + 时间距离估计**

### 2.2 模型框架
模型主要分为三阶段：
1. **视觉特征提取**：通过共享 CNN 对观测序列与目标图像提取特征；
2. **时空融合模块**：
   - *空间部分*：采用动态图卷积（Dynamic Axial Graph）捕捉各图像区域之间的方向关系；
   - *时间部分*：通过混合 Temporal Shift 和多尺度差异感知卷积建模多帧间的动态；
3. **任务头**：
   - 输出控制动作（通过扩散模型反向去噪生成）；
   - 输出到目标的时间距离估计值。

### 2.3 核心技术细节

#### （1）动态轴向图构建
- 将每个图像特征图视作节点网格；
- 沿水平/垂直方向通过可学习温度参数 τ 构造软边；
- 利用“多尺度对比增强”获取层次结构特征；
- 残差连接与对比掩码防止过平滑。

#### （2）时间特征融合
- **Hybrid Temporal Shift**: 通过前向、后向、双向通道偏移（比例ρ）捕获短期时序依赖；
- **Multi-resolution Contrast**: 通过多尺度池化与图块旋转放大运动变化；
- **Contrast-aware Fusion**: 拼接差分特征后进行卷积融合，生成最终时空表示。

#### （3）联合训练损失
- **动作扩散损失**：基于Gaussian噪声扰动的去噪重建；
- **距离回归损失**：最小化预测与真实时间距离间的均方误差；
- 综合损失：
  \[
  L_{\text{total}} = \alpha L_{\text{dist}} + (1 - \alpha)L_{\text{diff}}
  \]
  其中 α 控制两任务平衡。
