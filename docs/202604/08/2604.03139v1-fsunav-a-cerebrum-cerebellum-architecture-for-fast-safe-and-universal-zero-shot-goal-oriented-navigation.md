## 摘要
当前的视觉-语言导航方法在异质机器人兼容性、实时性能和导航安全性方面面临显著瓶颈。此外，它们在开放词汇的语义泛化和多模态任务输入的支持上也存在困难。为解决这些挑战，本文提出了 FSUNav：一种用于快速、安全和通用零样本目标导向导航的“Cerebrum-Cerebellum”（大脑-小脑）架构，该架构创新性地将视觉-语言模型（VLMs）与所提出的结构相结合。小脑模块是一个高频端到端模块，基于深度强化学习构建通用局部规划器，使异质平台（例如类人机器人、四足机器人、轮式机器人）能够实现统一导航，在提升导航效率的同时显著降低碰撞风险。大脑模块构建了一个三层推理模型，并利用 VLMs 建立端到端的检测与验证机制，实现无需预定义 ID 的零样本开放词汇目标导航，从而在仿真与真实环境中提高任务成功率。此外，该框架支持多模态输入（例如文本、目标描述和图像），进一步提升了泛化能力、实时性能、安全性和鲁棒性。在 MP3D、HM3D 和 OVON 基准测试上的实验结果表明，FSUNav 在物体、实例图像和任务导航上均取得了最先进的性能，显著优于现有方法。在多种机器人平台上的实际部署进一步验证了其鲁棒性和实用性。

## Abstract
Current vision-language navigation methods face substantial bottlenecks regarding heterogeneous robot compatibility, real-time performance, and navigation safety. Furthermore, they struggle to support open-vocabulary semantic generalization and multimodal task inputs. To address these challenges, this paper proposes FSUNav: a Cerebrum-Cerebellum architecture for fast, safe, and universal zero-shot goal-oriented navigation, which innovatively integrates vision-language models (VLMs) with the proposed architecture. The cerebellum module, a high-frequency end-to-end module, develops a universal local planner based on deep reinforcement learning, enabling unified navigation across heterogeneous platforms (e.g., humanoid, quadruped, wheeled robots) to improve navigation efficiency while significantly reducing collision risk. The cerebrum module constructs a three-layer reasoning model and leverages VLMs to build an end-to-end detection and verification mechanism, enabling zero-shot open-vocabulary goal navigation without predefined IDs and improving task success rates in both simulation and real-world environments. Additionally, the framework supports multimodal inputs (e.g., text, target descriptions, and images), further enhancing generalization, real-time performance, safety, and robustness. Experimental results on MP3D, HM3D, and OVON benchmarks demonstrate that FSUNav achieves state-of-the-art performance on object, instance image, and task navigation, significantly outperforming existing methods. Real-world deployments on diverse robotic platforms further validate its robustness and practical applicability.

## 二、方法论：FSUNav 框架

### 1. 总体思想
- 模仿人类神经系统的分工：  
  **「Cerebrum（大脑）」用于高层语义理解与推理**；  
  **「Cerebellum（小脑）」用于低层运动控制与实时避障**。
- 两模块协同工作，实现零样本、跨平台的目标导向导航。

### 2. 架构组成
**(1) Cerebellum 模块 — 通用局部规划器**
- 基于 **深度强化学习（DRL）** 的平台解耦导航控制器。
- 输入仅依赖局部传感信息（如深度、LiDAR、速度状态）。
- 输出即时控制指令 \( u_t = (v_t, \omega_t) \)，用于避障和目标导向移动。
- **关键技术**：
  - 采用 **PointNet** 网络处理几何点集输入。
  - 将机器人形状参数（长宽、高速极限、加速度极限）嵌入点集特征中，形成 **dimension-aware 表征**。
  - 提出 **课程学习** 策略：按机器人体积分组，逐步增强训练场景难度，提升可迁移性。
- **核心效果**：实现真正的跨平台泛化，无需重新收集训练数据或微调。

**(2) Cerebrum 模块 — 三层高层推理系统**
- 构建 **三层结构：语义层（Semantic）、空间层（Spatial）、规则层（Rule）**。
  - **语义层**：由 **Vision-Language Model (VLM)** 实现，解析输入（文本、图像或描述）为统一目标表示 \( P_{target} = \langle c_{core}, A_{attr}, R_{ctx} \rangle \)，并生成目标物体的检测框。
  - **空间层**：结合 **几何占据图（BEV Map）** 与 **VLM指导的语义子目标生成**，在目标不可见时由模型选择“安全且语义相关”的探索点。
  - **规则层**：负责任务决策与验证；通过 **双阶段检测**：
    1. 全局场景验证目标是否存在；
    2. 局部匹配候选区域与目标属性，输出验证函数 \( V(c,G) \in \{0,1\} \)。
  - 引入 **冷却机制 (cooldown)** 避免频繁计算，提升实时性。
  - VLM 同时生成 **语义场景图**（关系如“在上”、“旁边”等），提升推理层理解。

**(3) VLM 的核心作用**
- 不仅充当视觉检测器，更作为 **语义推理引擎**：
  - 执行目标解析、条件检测、子目标生成、场景验证、语义关系构建。
  - 实现 **开放词汇零样本导航** — 无需预定义类别或训练集映射。

## 三、实验设计

### 1. 数据集和任务
- 使用三个主要 **基准数据集**：
  - **Matterport3D (MP3D)**
  - **Habitat-Matterport3D (HM3D)**
  - **Open-Vocabulary Navigation (OVON)**
- 对应三类任务：
  1. Object-Goal Navigation (ON)
  2. Instance-Image Navigation (IIN)
  3. Text-Goal Navigation (TN)
- 同时测试开放词汇目标（OV）。

### 2. 实验指标
- **Success Rate (SR)**：成功率，判断是否在目标 1m 范围内停下。
- **Success weighted by Path Length (SPL)**：路径效率指标，衡量成功与最优路径比。

### 3. 对比基线
涵盖四大类方法：
- **监督学习方法**：ZSON、PEANUT+LOG、IEVE 等。
- **通用监督方法**：PSL、GOAT。
- **零样本方法**：ESC、SG-Nav、CogNav、REST 等。
- **通用训练免费方法**：UniGoal。
该组合覆盖训练型、零样本型、通用泛化型的主流范式，确保对比公平。
